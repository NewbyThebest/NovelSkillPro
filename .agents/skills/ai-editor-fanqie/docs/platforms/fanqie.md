# 番茄平台（番茄小说 / 免费投流向）审稿指南

本文件是针对"番茄小说"分发平台的专属审美与避坑标准，作为脚本或网络失败时的兜底数据。执行审稿时优先运行 `scripts/fetch_fanqie_rank.py` 获取或读取缓存的番茄网页版公开榜单数据，并可结合用户提供的番茄 App 榜单截图或观察校验；本文件只用于网络失败、接口异常或需要快速理解番茄男频长篇口径时。

## 风口判断口径提醒

* **榜单脚本优先**：风口判断优先使用 `scripts/fetch_fanqie_rank.py` 抓到或缓存的番茄网页版分类榜、在读榜、新书榜。
* **30 天缓存优先**：脚本默认将同一题材、性别、榜型、数量的结果记录到项目 `AI 责编/榜单缓存/`，30 天内复用，不重复请求。报告中引用缓存时要带上 `fetched_at` 日期。
* **强制刷新条件**：用户明确说“最新”“今天”“重新查榜”“强制刷新”时，运行命令追加 `--refresh`；若只是普通审稿，优先用一个月内缓存。
* **废弃第三方标签主口径**：不再用第三方 potential_score 作为平台匹配度主评分；第三方数据只能作为用户特别要求时的附录参考。
* **榜单头部验证必须参与修正**：若某题材在巅峰榜、阅读榜或网页版分类榜 TOP10/TOP30 中密集出现，应上调赛道判断。
* **优先使用本技能脚本查榜**：运行 `scripts/fetch_fanqie_rank.py` 获取番茄网页版公开排行榜数据。常用命令：
  * `python scripts/fetch_fanqie_rank.py --list-categories`
  * `python scripts/fetch_fanqie_rank.py --category 都市高武 --gender male --rank-type read --limit 30 --summary`
  * `python scripts/fetch_fanqie_rank.py --category 都市高武 --gender male --rank-type read --limit 30 --summary --refresh`
  * `python scripts/fetch_fanqie_rank.py --scan --gender male --rank-type read --per-category 5 --max-categories 12 --summary`
* **字体反爬说明**：番茄网页版会用 PUA 私用区字符和字体混淆部分书名/简介。当前脚本不强制解码字体；做风口判断时优先用分类、排名、在读人数、书籍链接和可读字段，精细分析书名简介时再考虑增加字体解码。
* **冲突时必须明示**：报告里要说明“榜单脚本数据”和“用户提供的 App 榜单/公开资料”是否一致；不得把第三方标签接口包装成番茄官方风向。
* **无法核验榜单时要声明**：不能访问实时榜单时，必须写明“未能核验番茄 App 实时榜单”，并降低风口结论的确定性。

## 核心受众偏好
* **在线用户 + 短耐心**：番茄核心读者是碎片化阅读的免费用户，首屏 30 秒决定生死，第一章前 200 字必须有钩子。
* **投流驱动**：番茄不是作品驱动的平台，而是投流组驱动的平台。好稿子能拿到投流资源，才能爆。**选题、书名、简介、开篇必须"投流友好"**。
* **套路吃透 + 密度拉满**：番茄读者**不反感套路**（战神、赘婿、真假千金、系统降临、签到、重生、神豪），但要求**夸张化、密度化、低延迟兑现**。
* **节奏包容度为零**：零铺垫，零说明文，零慢热。金手指最迟第二章兑现。
* **短视频物料优先**：每一章都要考虑"这段能不能剪 15 秒短视频投流"。
* **题材匹配的人物关系优先**：暧昧/多女主是番茄男频重要热门赛道，但不能全题材一刀切。都市、神豪、文娱、校园、乡村、日常向作品重点看女主/暧昧张力；高武、玄幻、历史、官场同人、悬疑、硬科幻等题材重点看战斗升级、权力压迫、宿敌、师徒、队友或反派关系爆点。

**标签介绍**：男主角与 3 位以上女性角色产生深度情感、利益或肉体纠葛的男频核心题材。背景可以是都市、乡村、玄幻、修真。核心金手指多为多子多福、神豪、签到、神医等传承系统。本质是满足读者的收集欲与征服欲。

## 番茄避坑铁律

1. **慢热开局 = 死刑**：第一章前 200 字无钩子，直接扑街。
2. **说明文开篇 = 死刑**：用户没耐心看世界观介绍。
3. **虐主超一章 = 毒抗崩溃**：番茄读者毒抗远低于起点。
4. **金手指蓄力 = 节奏崩盘**：最迟第二章兑现第一次爽感。
5. **人物关系没爆点 = 投流素材变弱**：暧昧/多女主、神豪、都市情感等题材女主不上会严重扣分；非情感核心题材则必须尽早建立宿敌、反派、师徒、队友、家族压迫等可剪辑的人物关系爆点。
6. **内心戏过多 = 文青病**：对话驱动 + 动作驱动 + 反应驱动，少用独白。
7. **长句长段 = 阅读疲劳**：单句 ≤ 20 字，单段 ≤ 4 行。
8. **严格单一视角 = 浪费机会**：番茄鼓励切反派视角、路人视角强化爽感。
9. **书名不够抓 = 投流组不收**：书名必须 10-15 字内具备强信号。
10. **没考虑前期留存爆点 = 追读流失**：30-50 章必须安排一次大爆点支撑追读、完读和投流素材复用。
