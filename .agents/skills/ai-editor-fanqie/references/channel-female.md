# 番茄女频频道增量口径

仅在 `CHANNEL=female` 时读取，用于补充共用口径，不改变评分结构。具体题材按 `genre` 路由。

## 频道边界

- 女频不等于言情、甜宠或细腻文风。事业成长、升级战斗、系统脑洞、悬疑探案、种田经营、家国群像和无感情线作品都可以成立。
- 不强制系统、外挂、当场打脸，也不强制男主在第一章出场。只检查作品承诺的核心人物、关系、目标和问题是否及时建立。
- 关系与家庭可以成为主线，不必为了证明主角独立而强行削弱这些内容；但核心选择不能长期由伴侣、家人或系统代替完成。
- 作品没有承诺感情线时，不得因缺少暧昧、甜宠、虐恋或男女主互动而扣分；作品承诺多条主线时，再检查各线是否真正参与推进。

## 频道增量校准

### 故事主线

- 检查核心主角是否拥有可指认的处境、目标、筹码和选择，以及每个阶段是否改变关系、身份、事业、家庭利益、地位、安全处境、能力或认知。
- 关系线、事业线、斗争线、悬疑线和成长线都可以成为主轴。只按作品实际承诺检查，不预设必须双线并行。
- 高压关系可以成立，但压力必须推动边界、选择、反制、离开、合作、和解条件或局势变化。

### 前三章节奏

- 开篇可以从命运转折、关系变化、身份变化、利益受损、职业危机、家庭冲突、升级机会或异常事件切入。
- 允许情绪蓄力和关系铺垫，但期间必须产生观察、判断、选择、布局、信息变化或关系变化。
- 第三章前应出现一次与题材承诺相符的进展，不强制必须当场反击、进入恋爱或完成身份逆袭。

### 核心梗与设定

- 关系结构、身份差、职业处境、家族结构、重生穿越规则、能力机制、斗争目标、经营目标或悬疑异常，都可以构成核心梗。
- 重点检查设定是否持续改变人物选择、关系和局势，而不是只在简介中形成标签。
- 熟悉套路可以依靠更具体的现实痛点、人物立场、职业细节、关系结构或能力规则形成新切口。

### 文笔与代入感

- 不强制细腻抒情、身体反应密集或大量心理独白。轻松、锋利、克制、热血、悬疑和生活流风格都可以。
- 情绪变化要有明确诱因，并通过动作、选择、对话、利益和关系结果表现；不能只靠旁白宣布爱恨。
- 人物互动是否改变关系和局势，比单纯描写“氛围感”更重要。

## 女频常见错配风险

以下风险必须结合具体题材和作品承诺判断，不是所有女频作品的固定扣分项：

- **核心选择被代办**：主角长期由伴侣、家人、系统或配角替她决定和解决核心问题。
- **关系高压却不结算**：作品主打关系冲突时，伤害不断叠加，却没有边界、反制、离开、合作条件或关系变化。
- **关系缺少成立依据**：爱、恨、牺牲和忠诚缺少共同经历、利益、价值观或处境支撑。
- **女性配角工具化**：女性配角只为争夺男性、衬托主角或承担恶意，没有自己的利益和判断。
- **把力量型题材误判为男频**：女主升级、战斗、经营、权谋或使用系统本身不构成频道错配，仍应按目标读者承诺和题材执行判断。

## 题材职责边界

- 感情、职业、礼法、斗争、线索和升级方式由对应题材 reference 判断；本文件只检查女频频道承诺，不强行套用恋爱双线。
- 年代、种田没有独立 reference 时，按核心主线选择现言、世情、历史、都市或脑洞中最接近的一项。
