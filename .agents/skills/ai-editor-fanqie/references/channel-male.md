# 番茄男频频道增量口径

仅在 `CHANNEL=male` 时读取，用于补充共用口径，不改变评分结构。具体题材按 `genre` 路由。

## 频道边界

- 男频不等于系统、后宫、搞钱或当场打脸。无系统、单女主、无女主、关系向、家国群像、历史、悬疑、种田和现实题材都可以成立。
- 男频常见追读承诺集中在目标推进、行动结果、能力或资源变化、身份与秩序变化、事业扩张、求生、探索真相等方向，但不要求一本书同时具备这些元素。
- 情感、家庭、责任和人物关系也可以成为主线。只要作品承诺清楚并持续推进，不得因为收益不是“变强、赚钱、装逼”而扣分。
- 阶段回报可以是实力、资源、身份、权力、事业、知识、名望、生存空间、关系确认、责任完成或真相推进。

## 频道增量校准

### 故事主线

- 检查主角是否有可持续推进的目标，以及每个阶段是否改变资源、能力、身份、局势、关系或认知。
- 历史、悬疑、群像和现实题材不必使用升级打怪结构，但必须有阶段任务、进展和后续发动机。
- 强身份或强能力只有转化成选择、代价和剧情结果时才算有效证据。

### 前三章节奏

- 开篇可以从危机、竞争、身份反差、利益目标、责任压力、异常事件、谜团或关系变化切入。
- 作品有金手指时，检查其核心作用是否及时进入剧情；没有金手指不扣分，只检查作品自己的核心卖点。
- 第三章前应出现一次与题材承诺相符的进展，不强制必须是升级、打脸或物质收益。

### 核心梗与设定

- 能力机制、身份反差、职业优势、信息差、历史处境、调查目标、责任冲突或世界规则，都可以构成核心梗。
- 重点检查核心梗能否持续制造任务、选择、冲突和结果，不以“爽点是否粗暴”替代长期可写性。

### 文笔与代入感

- 不强制市井气、粗粝感、搞笑吐槽或短句连击。轻松、热血、冷峻、克制、悬疑和群像风格都可以。
- 只检查信息是否清楚、行动是否可追踪、结果是否可见，以及文风是否服务作品自己的读者承诺。

## 男频常见错配风险

以下风险必须结合具体题材和作品承诺判断，不是看到关键词就扣分：

- **主角承诺与行动脱节**：作品主打成长、事业、求生或调查，主角却长期不采取有效行动，也没有形成策略。
- **阶段变化缺失**：反复遭遇相似冲突，但资源、身份、关系、认知和局势始终不变。
- **对手降智送结果**：对手放弃基本利益和判断，只为让主角轻易获胜。
- **多女主工具化**：作品明确采用多女主路线时，女性角色只有外貌、倒贴或提供资源的功能，没有自身处境、利益和判断。
- **把克制误判为无爽点**：智斗、家国、群像或现实题材可以延迟外显回报，但必须持续提供信息进展、策略变化或阶段结果。

绿帽、背叛、受辱、资源被夺等情节只有在主角长期承受核心损失、缺少合理回应，并破坏作品既定承诺时才构成高风险；不得按关键词直接判定。

## 题材职责边界

- 史料、线索、金手指、升级和开篇方式由对应题材 reference 判断；本文件只检查男频频道承诺。
- 种田没有独立 reference 时，古代经营按历史、现代生活经营按都市校准。
