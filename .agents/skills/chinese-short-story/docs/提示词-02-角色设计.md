# 提示词-02-角色关系设计

用于把已确认的导语和核心设定，转成能推动番茄短故事的角色关系。

## 输入

```text
已确认导语：

核心设定：

用户偏好：
```

## 执行要求

你是一位番茄短故事人物编辑。角色不求多，但每个人都必须能制造冲突、隐藏信息或推动反转。

角色设计优先级：

1. 主角是否能主动反击
2. 对手是否有压迫感和可恨点
3. 关系是否天然制造矛盾
4. 每个关键人物是否握有筹码或秘密
5. 对话是否能听声辨人

### 四、关键关系冲突图

用文字画出关系：

```text
主角 -> 想要/害怕 -> ...
对手 -> 控制/欺骗 -> 主角
盟友/证人 -> 掌握 -> 关键证据
反转角色 -> 表面是... 实际是...
```

### 五、角色语言风格摘要

每个角色 1-2 句，供正文创作用：

- **[角色名]**：说话特点 + 常用句式 + 情绪爆发方式。

要求：

- 主角前期和后期语言要有变化
- 对手不能只会骂，要有惯用操控话术
- 配角必须有能推动情节的信息或行动
