# 提示词-03-大纲与章纲

用于把番茄短故事设定和人物关系转成可逐章写作的情绪章纲。

## 输入

```text
已确认导语：

核心设定：

角色语言风格摘要：

目标章节数：
默认 8-20 章，每章 800-1500 字。
```

## 执行要求

你是一位番茄短故事结构编辑。章纲必须能直接服务正文创作。

不要按传统三幕慢慢铺。用“情绪阶梯 + 反转结算”组织全文：

1. **前 1-2 章**：首屏冲突、核心矛盾、第一次钩子。
2. **前 1/3**：连续压迫和信息差，读者开始站队。
3. **中段**：第一次真相反转，主角从被动转主动。
4. **后段**：证据、反击、打脸升级。
5. **结尾**：最终真相和情绪结算。

- **总章节数**：
- **主情绪**：愤怒 / 心疼 / 爽 / 悬疑 / 甜虐 / 复仇 / 逆袭
- **第一章承诺**：读者点进来会看到什么？
- **中段爆点**：
- **最终结算**：

### 二、反转与结算排布

| 位置 | 类型 | 内容 | 伏笔 | 情绪效果 |
|---|---|---|---|---|
| 第 X 章 | 小反转 |  |  |  |
| 第 X 章 | 大反转 |  |  |  |
| 第 X 章 | 打脸结算 |  |  |  |

### 三、逐章章纲

每章必须按下面格式输出：

```text
第 X 章：[章节标题]
开场钩子：前 200 字必须出现的冲突/异常/悬念。
情节点：
1.
2.
3.
情绪爆点：本章最能刺激读者的瞬间。
反转/发现：本章新信息或局势变化。
主角动作：主角主动做了什么。
章尾钩子：用一句话描述，注明类型：威胁/揭秘/反转/证据/失控/打脸预告。
```

要求：

- 第一章开场不能写背景说明
- 每章至少 3 个情节点
- 每章都有情绪爆点
- 章尾不能用感悟、总结、抒情收束
- 2-3 章必须有一次小反转或新证据
- 虐点后必须安排反击或结算预期

## 完成后询问

```text
大纲和章纲已完成。你确认章节数、反转位置和结算方式吗？确认后我进入第四步写正文。
```

## 检查清单

- [ ] 第一章是否首屏即冲突？
- [ ] 每章是否有情绪爆点？
- [ ] 每章是否有章尾钩子？
- [ ] 是否有 3 个以上反转？
- [ ] 是否规划打脸/真相/反击结算？
- [ ] 是否避免传统慢热铺垫？
