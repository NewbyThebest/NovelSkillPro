# 动作过程呈现

用于解决关键攻防只有结果、读者看不出双方怎样完成交锋的问题。它只补足必要过程，不负责把整场战斗逐招展开。

## 先判断是否展开

- 动作没有改变位置、姿势、伤势、能力信息、战术或胜负趋势时，概写即可。
- 关键命中、失误、反制和决定胜负的交锋需要读者看懂原因时，再展开过程。
- “一脚踢飞”“挥刀逼退”等直写可以用于过渡、清理普通对手或结果本身已经足够清楚的动作。
- 第一次展示能力不等于必须写细，只展开会影响读者理解和后续判断的部分。

## 关键交锋

围绕一个尚未解决的攻击、控制或脱身问题展开特写。前一个动作还没有完成，或者它造成的接触、姿势和力量仍在限制双方时，后续变招属于同一次交锋。

允许双方连续反制，但每次变化都要改变当前威胁、接触位置、借力条件或可选应对。防守可以同时反攻，变招可以发生在上一招尚未结束时，避免写成一方完成动作后另一方再重新起招。

双方重新站稳、拉开距离、更换攻击目标或已经解决当前威胁时，结束这次特写。没有改变攻防关系的换手、试探和重复动作直接概括。

## 因果解释

动作本身无法说明下一步为什么发生时，直接解释最重要的一项：人物想达到什么、时间或环境在逼迫什么、当前姿势和伤势限制了什么，或者人物根据什么作出判断。

- 把解释放在选择发生之前或变招之中，让它成为下一步动作的原因。
- 动作意图和限制已经一眼可见时，不额外解释。
- 不复述已经写出的动作结果，不逐项讲解肌肉、发力和招式步骤。
- 设定能力第一次影响交锋时说明必要规则，后续只写这条规则怎样改变当下选择。

## 动作落地

根据当前动作按需补足发力、方向、接触或受力反馈，只写读者理解这次交锋所需的信息，不把这些内容当成固定清单。

- 用视觉交代攻击从哪里来、逼向哪里，以及双方距离和姿势怎样变化。
- 用声音说明速度、碰撞或现场物体受力，声音必须有实际来源。
- 用触觉和身体平衡表现震动、麻木、失衡、呼吸受阻或伤势对下一步的影响。
- 嗅觉和味觉只有在现场确有来源并影响判断时才使用。

一次关键交锋通常选择最有用的一两类感官信息。动作已经清楚时，不为增强画面继续添加声音、环境反应或身体反馈。

## 详略切换

- 用概写跨过重复试探、追击和无效换招，把篇幅留给真正改变局面的交锋。
- 只特写会改变当前攻防问题的动作；姿势不同但威胁和应对没有变化的部分压成过渡。
- 允许一段概写后进入一次特写，也允许关键交锋结束后立即恢复概写，不按固定比例分配篇幅。
- 低境界近身战优先保持动作和受力可信；境界、武核或兵器能力提高后，可以适量增加声势和意象，但不能盖住动作结果。
- 比喻只在比直写更容易看懂形态、速度或力量时使用；环境反应要符合现场和力量层级。
- 结果已经通过人物、兵器或环境变化落地后，不再旁白复盘整场交锋。

## 自查

- 这次动作是否真的值得详细展开？
- 读者能否看清攻击想做到什么，以及对手怎样应对？
- 连续变招是否都在处理同一个尚未解决的问题，而非轮流重新起招？
- 关键选择的原因是否清楚，解释是否只补了必要信息？
- 感官和环境细节是否提供了新信息，而不是为了凑画面？
