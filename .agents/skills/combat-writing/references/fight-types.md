# 战斗类型模板

按战斗类型选模板。一个场景可以叠加两种类型，例如“规则擂台 + 藏拙”“追杀 + 反杀”“越级战 + 强敌试探”。

## 规则擂台

适用：比武、训练考核、擂台赛、积分战、陪练测试、宗门试炼。

关键要素：

- 规则必须变成压力：倒计时、扣分、越线、禁止逃跑、必须反击、不能杀人、限定武器。
- 裁判/系统/教官负责结算爽点，不一定需要围观者。
- 对手可以利用规则逼主角犯错。
- 主角可以钻规则缝隙完成反击、藏拙、控分或保命。

节拍：

```text
规则公布或暗示 -> 对手用规则压人 -> 主角被迫暴露一部分能力 -> 反击得分/避开惩罚 -> 对手升级 -> 主角在规则边缘完成目标 -> 系统或裁判结算。
```

## 追杀反杀

适用：巷战、山林逃亡、夜袭、逃课式试炼、被围堵。

关键要素：

- 路线比招式重要：门、窗、墙、楼梯、树、河、废墟、盲角。
- 呼吸和体力是倒计时。
- 反杀不要突然，应提前埋可利用的地形、道具或误导。

节拍：

```text
危险预警 -> 被迫逃/边打边退 -> 伤势和体力恶化 -> 利用地形制造错判 -> 对手冲过头 -> 主角反杀或脱身。
```

## 群战

适用：一对多、混战、护送、突围、守门、救人。

关键要素：

- 不要同时细写所有人，永远锁定主角视线中的一两个威胁。
- 用“方位 + 距离 + 优先级”写清局面：左侧刀、背后箭、正面盾、脚下阵。
- 群战爽点来自调度：借敌挡敌、打乱阵型、擒贼先擒王、制造缺口。

节拍：

```text
包围成形 -> 主角识别最大威胁 -> 先破一角 -> 敌人补位 -> 主角借位/借尸/借兵器扩大混乱 -> 打开出口或斩首。
```

## 越级战

适用：主角弱打强、境界差、资源差、重伤硬撑。

关键要素：

- 强者必须让读者感到压迫：一招就逼主角付代价。
- 主角赢不能靠对手降智，要靠情报、地形、规则、弱点、代价、底牌。
- 胜利最好是局部胜利：撑过一招、伤到对手、夺到物品、拖到时间、保住人。

节拍：

```text
强者压场 -> 主角第一次吃亏 -> 发现规律/弱点 -> 用代价换机会 -> 短暂反制 -> 目标达成但留下后患。
```

## 秒杀战

适用：立威、打脸、清杂兵、展现实力差。

关键要素：

- 秒杀前要铺垫对方嚣张、身份、战绩、旁人预期。
- 出手过程要短，结算要足。
- 爽点在“别人以为会怎样”和“结果突然反过来”。

节拍：

```text
对手放狠话 -> 众人默认主角会输/麻烦 -> 主角只动一次 -> 对手倒下 -> 结算身份差和后果。
```

## 斗法/异能战

适用：仙侠斗法、玄幻术法、异能、武核、阵法、法宝。

关键要素：

- 先定规则：能力代价、范围、触发条件、克制关系、冷却或消耗。
- 斗法不是光污染，要写“能力如何改变局面”。
- 用可见结果承载抽象力量：符纸燃尽、阵纹开裂、空气扭曲、地面结霜、金属失重。

节拍：

```text
能力亮相 -> 主角判断规则 -> 第一次应对失败或付代价 -> 找到克制/空隙 -> 对手加码 -> 主角用规则反制。
```

## 强敌试探

适用：主角暂时不能赢、boss初见、导师压迫、强敌测验。

关键要素：

- 目标不是胜利，而是获得情报、撑住时间、保护秘密、确认差距。
- 结尾可以输，但要让主角拿到下一阶段的钥匙。

节拍：

```text
强敌随手压制 -> 主角意识到差距 -> 用小技巧换一次喘息 -> 强敌认真一瞬 -> 主角受伤但读出信息 -> 为下一次战斗埋钩子。
```
