# 藏拙与控分模板

适用：主角不能暴露全部实力，不能赢太明显，需要假败、控分、装弱、保护同伴、避开组织关注、维持一明一暗格局。

## 核心矛盾

藏拙战不是“打不过”，而是“能赢但不能让别人知道能赢”。主角同时面对两个敌人：

- 明面敌人：正在打他的对手。
- 暗面敌人：系统、教官、组织、旁观高手、记录数据、未来审查。

## 写法重点

- 让主角先看见“赢的后果”，再选择不赢。
- 藏拙不能太假，必须有合理代价：受伤、扣分、错失机会、被误解、被惩罚。
- 反击要真实但收力：系统认定有效，高手略有疑惑，对手以为只是运气。
- 演弱不是犯蠢，要表现主角精细控制边界。

## 常用策略

- **假装力竭：** 拳到半路偏开，脚步乱，呼吸失控，但关键部位仍保护得很好。
- **以小伤换隐藏：** 故意吃擦伤，不暴露完整身法或底牌。
- **有效但不致命：** 反击打肩、肋侧、手腕外侧，只让规则通过，不让对手倒下。
- **借对手遮掩：** 把反击伪装成对手冲太猛、自己被撞倒、双方失衡。
- **控分控线：** 让系统给最低有效分，不触发高评价。
- **让对手背锅：** 对手狂化、越线、违规或体力耗尽，主角只是“撑住了”。

## 节拍模板

```text
对手压制 -> 主角发现可赢机会 -> 外部观察者/规则提醒风险 -> 主角选择收力 -> 付出小代价维持伪装 -> 规则判定勉强过关 -> 真正目标达成。
```

## 心理句式

- 赢很容易，赢完之后怎么办，才是麻烦。
- 这一拳落实，对手会倒，记录也会亮。
- 他要的不是胜，是刚好不输。
- 不能太亮，也不能太假。
- 明处已经有人站着了，他必须留在暗处。

## 结算方式

- 系统给出低评价但没有惩罚主角。
- 对手以为自己只是差一点，保留误判。
- 教官/高手记录异常，但暂时没有定论。
- 主角受伤离场，看似狼狈，实际保住秘密。
- 读者知道主角能赢，角色世界不知道，形成藏锋爽。
