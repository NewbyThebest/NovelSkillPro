# 番茄冲突设计

把“剧情有点平”改成番茄读者愿意继续看的冲突：首屏有事、人物有压迫、主角有动作、章尾有钩子、后续有结算。

## 适用场景

用于以下请求：

- “这段太平了，帮我加冲突”
- “设计一场矛盾/争吵/打脸/反转”
- “这一章爽点不够”
- “帮我做一个两难选择”
- “这个开头怎么更抓人”
- “按番茄节奏升级冲突”

如果用户要的是纯文学、慢热群像、严肃现实主义，先说明本技能默认按番茄网文节奏处理。

## 核心原则

1. **首屏即压迫**：场景前 200 字内必须出现问题，不能先铺背景。
2. **冲突来自人和利益**：不要只有“我很难受”，要有某个人、规则或事件正在抢主角的东西。
3. **读者情绪要明确**：愤怒、心疼、好奇、爽、恐惧、期待，至少命中一个。
4. **主角必须行动**：主角可以暂时吃亏，但必须观察、试探、反击、取证、设局或做选择。
5. **冲突必须改变局面**：结束后资源、关系、信息、地位、危险等级至少变一项。
6. **虐点要配结算预期**：可以压主角，但要给读者看到后面怎么反杀、揭穿或打脸。

## 番茄冲突六型

按场景选择 1-2 个叠加：

| 类型 | 核心刺激 | 适用场景 | 关键问题 |
|---|---|---|---|
| **羞辱压迫** | 主角被当众踩脸 | 退婚、职场、家族、校园、豪门 | 谁让主角下不来台？ |
| **利益抢夺** | 钱、房、资源、名额、功劳被抢 | 世情、都市、玄幻、末世 | 对方拿走了主角什么？ |
| **关系背叛** | 亲人/爱人/朋友反咬 | 婚恋、宅斗、团队、兄弟 | 背叛证据是什么？ |
| **规则倒计时** | 不做选择就会损失 | 悬疑、末世、系统、竞技 | 主角还剩多少时间？ |
| **秘密揭露** | 一个新信息改写局面 | 真假千金、重生、悬疑、权谋 | 揭露后谁会崩？ |
| **强者碾压** | 强敌或权力压迫主角 | 玄幻、高武、历史、朝堂 | 主角凭什么破局？ |

## 冲突升级公式

用这 6 步设计或改写冲突：

1. **主角要什么**：本场景主角的具体目标。
2. **谁来抢/挡/骗/逼**：阻碍方必须有自己的利益。
3. **压迫动作**：对方当场做了什么，而不是只说了什么。
4. **主角代价**：不反击会失去什么？反击会付出什么？
5. **主角动作**：主角如何观察、试探、反击、取证、设局或硬刚。
6. **局面变化**：冲突结束后出现新证据、新敌人、新危险、新爽点或新钩子。

## 改稿模式

如果用户提供了原片段，先判断问题类型：

- **没压迫**：对手只是说话，没有实际抢夺/威胁。
- **没代价**：主角输赢都无所谓。
- **没动作**：主角只想不做。
- **没升级**：整场都在重复同一个情绪。
- **没钩子**：章尾收成感悟或解释。

然后输出：

```markdown
### 原问题
[指出 2-4 个具体问题]

### 升级方案
[按番茄冲突六型重排]

### 改写片段
[给出可直接替换的片段]
```

## 题材校准

- **世情/家庭**：利益抢夺 + 关系背叛最有效，重点写“理所当然的剥削”。
- **都市/职场**：羞辱压迫 + 利益抢夺，重点写身份反差和证据反杀。
- **玄幻/高武**：强者碾压 + 规则倒计时，重点写境界压迫和即时突破。
- **末世**：规则倒计时 + 利益抢夺，重点写物资、生存名额、安全区。
- **悬疑/灵异**：秘密揭露 + 规则倒计时，重点写异常、线索、误判。
- **古言/宫斗/宅斗**：关系背叛 + 秘密揭露，重点写身份、礼法、证据链。
- **现言/追妻/豪门**：关系背叛 + 羞辱压迫，重点写误会证据和情绪反噬。

## 自查清单

- [ ] 前 200 字是否有冲突或异常？
- [ ] 阻碍方是否有明确利益？
- [ ] 主角不行动是否会付出代价？
- [ ] 主角是否做了主动动作？
- [ ] 本场是否有情绪爆点？
- [ ] 结尾是否留下反转、证据、威胁或打脸预告？
- [ ] 冲突结束后局面是否发生变化？
