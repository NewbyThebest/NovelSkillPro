# 番茄脑洞题材

把番茄课堂里的 `男频脑洞 / 女频脑洞 / 副本设计 / 脑洞避坑` 合并成一个脑洞专用技能。

## 硬触发边界

只在下面情况使用：

- 系统流
- 高概念金手指
- 规则怪谈 / 惊悚副本
- 快穿、无限、位面、穿书等强设定脑洞
- 一句话卖点很强的高概念网文

## 脑洞文先抓核心脑洞

先让用户用一句话说清：

`主角拿到什么离谱设定 -> 这个设定怎样持续制造爽点`

如果一句话说不清，说明脑洞还没立住。

## 题材判断

- 男频脑洞：升级、反杀、系统兑现、规则破解
- 女频脑洞：身份反差、情感代偿、位面任务、萌宝/玄学/星际等
- 副本脑洞：规则、通关、资源、惩罚、反转

## 书名与开篇

脑洞文书名优先承担两个作用：

1. 设置期待感
2. 筛选读者

开篇硬要求：

- 快速进设定
- 不长篇解释世界观
- 尽快展示脑洞第一次兑现
- 主线清楚

## 副本设计

副本至少要有：

- 规则
- 死亡代价
- 通关目标
- 资源或奖励
- 递进变化

不要只写诡异场面，不写玩法。

## 常见问题

- 脑洞新，但逻辑不自洽
- 金手指只响一声，后面没持续玩法
- 书名没把最强卖点抛出来
- 开篇先解释设定，迟迟不爆点
- 副本只有皮，没有规则和回报

## 输出方式

1. 用一句话重压脑洞卖点
2. 判断更偏男频 / 女频 / 副本
3. 设计第一轮兑现
4. 补书名和开篇
