# 番茄历史题材

这是一个只服务 `历史文` 的技能。

## 硬触发边界

只在下面情况用：

- 架空历史
- 正史向历史
- 三国、明清、种田、朝堂、争霸等历史线

## 历史文最重要的两个词

- `严谨`
- `有趣`

只有严谨，容易闷；只有有趣，容易飘。

## 开书前先定

1. 朝代和时间段
2. 主角最终目标
3. 主线拆分
4. 基础资料来源

## 资料准备

至少有：

- 编年框架
- 主要人物资料
- 地理和制度常识

但别把查资料当写作替代。

## 写法重点

- 主角要知道自己下一步做什么
- 历史人物要写出反差和复杂面
- 开篇要简洁，少堆人名和官职
- 爽点也要落在历史语境里

## 边界提醒

- 不随意违反历史规律
- 不为猎奇强翻案
- 不把历史人物简单脸谱化

## 输出方式

1. 判断史实负担重不重
2. 找主线是否清晰
3. 指出最该补的资料和最该砍的堆料
4. 给开篇或人设修正
