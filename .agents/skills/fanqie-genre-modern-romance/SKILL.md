# 番茄现言题材

这是一个只服务 `现代言情` 的技能。

## 硬触发边界

只用于：

- 现代言情
- 豪门、娱乐圈、校园、婚恋、真假千金
- 女主职业成长文

## 现言先看双线

现言最怕只剩恋爱，没有剧情；也怕只剩剧情，没有情感牵引。

默认都要看两条线：

1. `感情线`
2. `剧情线`

## 好用的起手方式

找人物命运开始转折的节点做开篇，比如：

- 被迫结婚
- 身份曝光
- 一夜情后续
- 进新行业
- 回归原家庭

## 题材和套路怎么用

番茄课更强调：

- 在成熟套路里找新切口
- 换视角、换身份、换关系角色
- 保留读者熟悉的情绪预期

## 女主职业成长

职业文至少要有：

- 清晰行业
- 晋升路径
- 职业技能发挥场景
- 行业矛盾

不能只是披着职业皮谈恋爱。

## 常见问题

- 男女主刚见面就空心心动
- 只有甜虐，不见主线推进
- 女主职业只是标签，不参与剧情
- 套路都对，但切入点太旧

## 输出方式

1. 判断主打哪一条感情 / 剧情组合
2. 找开篇转折点
3. 补女主成长线
4. 补悬念和期待目标
