# 番茄新媒体文

这是一个只服务 `新媒体导向` 小说的技能。

## 硬触发边界

只在下面情况用：

- 明确追求投放转化
- 噱头、前几章、付费点是主要优化目标
- 作品本身就偏赘婿、战神、虐文、强情绪引流型

如果只是普通都市或现言，不要强行套新媒体逻辑。

## 新媒体文的核心目标

不是“写得完整”，而是把读者一路推过这几道门：

1. 愿意点
2. 愿意看前几章
3. 愿意看完免费期
4. 愿意为卡点付费

## 开篇和免费期

番茄课给出的重点是：

- 开篇有噱头
- 切入快
- 情绪拉满
- 免费期几万字尽量一环扣一环

不重要的内容尽量后放：

- 复杂设定
- 慢节奏成长
- 冗长修炼 / 背景补课

## 计费点设计

转付费处必须留：

- 更大的真相
- 更重的危机
- 更强的反击前夜
- 更足的情绪未结算

不要把付费点放在“刚把事情说清”那里，要放在“读者差一步就吃到回报”那里。

## 常见问题

- 噱头有了，但承接不住
- 免费期节奏松，读者还没上头就掉了
- 情绪一直高喊，但没有事件推进
- 卡点不够值钱，读者不愿付费

## 输出方式

1. 判断是否真适合新媒体
2. 拆开篇噱头
3. 拆免费期链路
4. 重做计费卡点
