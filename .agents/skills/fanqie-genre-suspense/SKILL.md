# 番茄悬疑题材

这是一个只服务 `悬疑 / 推理 / 灵异悬疑` 的技能。

## 硬触发边界

只用于：

- 推理探案
- 灵异风水
- 规则悬疑
- 悬爱
- 短篇悬疑氛围强化

## 先判断类型

1. `灵异风水`：气氛、因果、信息差更重要
2. `推理探案`：线索、逻辑、真凶迷惑性更重要

## 悬疑的核心

不是一堆怪事，而是：

`未知 -> 怀疑 -> 线索 -> 推进 -> 解释或反转`

## 写法重点

灵异风水优先：

- 氛围
- 信息不对称
- 反差环境
- 主角成长

推理探案优先：

- 离奇开篇
- 线索前后呼应
- 真凶不能太早露相
- 结局要意外且合理

## 合规和边界

- 不细写封建迷信宣传
- 不细写血腥和猎奇恶俗
- 不打擦边球
- 真实机关和现实政治要避开

## 常见问题

- 只有气氛，没有谜题
- 只有谜题，没有情绪牵引
- 真相全靠作者硬揭
- 线索断裂，前文埋的东西后面忘了

## 输出方式

1. 判断是灵异还是探案
2. 找悬念链断在哪
3. 补氛围或逻辑
4. 必要时重做开篇案件
