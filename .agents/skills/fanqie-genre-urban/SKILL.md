# 番茄都市题材

这是一个只服务 `都市` 的技能。

## 硬触发边界

只在下面情况用：

- 都市生活
- 女婿 / 战神 / 神医 / 奶爸等都市爽文
- 都市系统文

如果作品明确为投放导向的新媒体文，优先配合 `fanqie-genre-newmedia`。

## 先判断具体是哪类都市

1. `都市生活`：靠真实故事和共鸣
2. `战神 / 女婿`：靠身份压差和情绪爆点
3. `都市脑洞`：靠系统、反转、趣味感

## 都市生活怎么稳

- 先选故事，再定人称
- 开篇不要拖
- 第一章就把故事卖点展开
- 后续别只写恋爱拉扯，要补事业、友情、家庭和成长

## 女婿 / 战神怎么稳

- 小人物身份要真受压
- 强身份要真能改变局势
- 关系网要立体
- 冲突要围绕情绪和脸面展开

## 都市脑洞怎么稳

- 接地气
- 设定直给
- 系统兑现快
- 爽点密

## 常见问题

- 都市生活写成流水账
- 战神男主像空壳台词机
- 女婿冲突只有受辱，没有后续承接
- 系统文只有叮，没有故事

## 输出方式

1. 判断具体子类
2. 找当前最弱的一条线
3. 重压开篇卖点
4. 补关系网或升级链
