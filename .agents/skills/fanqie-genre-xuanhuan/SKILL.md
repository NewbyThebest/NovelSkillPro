# 番茄玄幻题材

这是一个只服务 `玄幻` 的技能，不负责泛题材判断。

## 硬触发边界

只有在下面情况才用：

- 传统玄幻
- 废柴逆袭
- 强者重生
- 玄幻脑洞
- 玄幻仙侠混合，但读者核心预期更像玄幻升级爽文

如果主轴是修仙求长生，优先转 `fanqie-genre-xianxia`。

## 先分两大路

1. `传统玄幻`：成长更慢，世界更大，升级更扎实
2. `脑洞玄幻`：系统更强，成长更快，套路更直接

不要一边想写传统玄幻，一边又把节奏写成短平快脑洞文。

## 玄幻最重要的不是“还能不能写”

番茄课的核心判断是：

不是传统玄幻不行了，而是 `同质化太重`。

所以先查三件事：

1. 主角底盘是不是太熟
2. 升级路线是不是一眼能猜到
3. 冲突是否还在用老一套废柴受辱模板

## 稳妥做法

- 保留读者熟悉的升级快感
- 在成熟框架里做微创新
- 让书名、开篇、主角能力点都带有效信息

可创新的位置优先是：

- 金手指兑现方式
- 主角身份反差
- 资源争夺规则
- 敌我关系
- 世界副本 / 地图结构

## 传统玄幻怎么写得更稳

- 升级体系要清楚
- 反派别全是降智垫脚石
- 主角要持续有目标，不只是挨打后反杀
- 每次升级都要换来更高层的地图、敌人或资源

如果写废柴逆袭：

- 羞辱桥段别重复换皮
- 每次反击都要改变主角地位
- 不要只会小打脸，不会推主线

## 输出方式

如果用户给的是设定：

1. 判断传统还是脑洞
2. 找最老套的一环
3. 给一处微创新
4. 补开篇卖点

如果用户给的是正文：

1. 判断同质化点
2. 找升级线是否清晰
3. 找反派 / 配角是否脸谱化
4. 给精修方向

## 与其他技能协同

- 开篇抓力不足，接 `fanqie-opening-hook`
- 单章节奏和爽点虚，接 `fanqie-pacing-conflict`
- 如果核心卖点已经变成系统脑洞，配合 `fanqie-genre-brainhole`
