# 番茄新书预期 (Initial Expectation)

> 在正式动笔或拆解样板书之前，请花几分钟思考你对这本番茄长篇的直觉预期。这不需要很精确，但能帮助你更好地掌控题材边界、读者情绪和商业卖点，而不是被 AI 推着走。

## 1. 题材边界 (Genre Boundaries)
- **核心类型**：(例如：玄幻、都市、科幻、仙侠...)
- **不想写什么**：(例如：不写后宫、不写系统、不写虐主...)
- **必须有什么**：(例如：必须有猫、必须有种田元素...)
- **核心爽点**：(例如：扮猪吃虎、人前显圣、通过智商碾压对手...)
- **番茄读者情绪**：(例如：逆袭、打脸、护短、复仇、救赎、甜宠、悬疑反转...)
- **投流卖点**：(一句话说清这本书最能被包装出去的噱头)

## 2. 预期篇幅与主线 (Length & Main Arc)
- **预期字数**：(例如：200万字、500章...)
- **一句话主线**：(例如：一个修车工成为星际霸主的故事)
- **前三章承诺**：(前三章要让读者看到什么爽点、关系爆点或金手指承诺？)
- **前30-50章留存爆点**：(例如：第一次大反击、身份揭露、关系反转、阶段性升级...)

## 3. 预期的设定、剧情与人设 (Plot & Characters)
- **主角形象**：(例如：腹黑、高智商、平时很怂关键时刻很勇...)
- **名场面脑洞**：(如果有想好的画面，可以写在这里)
- **预期设定**：(例如：金手指、世界观、人物关系等)
- **预期剧情**：(例如：主角如何获得金手指、如何利用金手指、如何升级金手指...)
- **预期人物**：(例如：主角、反派、配角等)
- **首屏切入点**：(第一章前200字准备用什么冲突、反差或危机抓人？)
- **章尾追读方式**：(更偏危机、反转、新收益、关系拉扯，还是秘密揭露？)

## 4. 个人执念 (Personal Obsessions)
- **为什么想写这本书？**：(例如：为了圆一个童年的梦、为了吐槽某个社会现象...)
- **对标作品**：(例如：想写一本像《诡秘之主》那样的小说...)
