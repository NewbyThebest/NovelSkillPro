# 番茄开篇钩子检查清单

来源：番茄课堂《开篇切入点的选择、剧情切入的关键要点》

## 1. 切入方式

### 重点判断

- 当前更适合剧情切入还是场景切入
- 用户是否有能力撑住场景切入

### 修改原则

- 新手默认剧情切入
- 场景切入只在氛围调度特别强时使用

## 2. 主线与卷纲

### 重点判断

- 整本主线能不能一句话说清
- 第一卷目标是否明确
- 主角当下为什么必须行动

### 修改原则

- 先有主线，再做切口
- 开篇服务主线，不服务作者炫设定

## 3. 剧情引爆

### 重点判断

- 开头是不是在“等事情发生”
- 作者有没有主动把炸点提前

### 命中雷点的典型信号

- 读了几百字还没真正出事
- 只是在交代过去，没有当前张力

### 修改原则

- 把最适合第一时间爆开的事提前
- 让读者不需要知道全部前情，也会被当下事件吸住
- 开篇每句话都尽量有功能，废话越少越好

## 4. 好奇升级链

### 重点判断

- 开篇是不是只有一个孤立钩子
- 有没有连续的小细节把读者往深处拖

### 修改原则

- 用 `小异常 -> 更异常 -> 引爆`
- 每一步都要让读者更好奇，而不是更迷糊
- 最好从第一个字开始就给读者一个抓手

## 5. 反差与锚点

### 重点判断

- 有没有一句话就能让读者记住的反差点
- 主角或场面有没有明确锚点

### 修改原则

- 反差越清晰，越容易抓眼
- 锚点要能快速形成角色记忆，不要虚

## 6. 角色控制

### 重点判断

- 前五章角色是否过多
- 是否有人纯粹凑热闹

### 修改原则

- 没作用的人直接删
- 每个角色出场都要服务主角、情绪、伏笔或后续剧情

## 7. 前三章节奏

### 推荐思路

1. 第一章：钩子 / 异常 / 当前危机
2. 第二章：叙事推进 + 伏笔升级
3. 第三章：第一次爆点 / 爽点 / 新坑

### 修改原则

- 每章结尾都尽量留钩子
- 前三章重点是让读者连着点下一章，不是把世界观讲完
- 三段话内要有重点，300 字内尽量进主题
