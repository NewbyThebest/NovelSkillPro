# 番茄节奏冲突爽点

这是一个面向 `单章 / 数章 / 单段剧情链` 的强化技能，重点不在于判断“有没有问题”，而在于把问题改成可读、可追、可爽的版本。

它主要解决五类高频问题：

1. 节奏混乱
2. 冲突乏力
3. 爽点不足
4. 过渡章发虚
5. 单章数据一掉就不知道怎么改

## 适用场景

- 这章太平了，帮我提速
- 这段冲突不够，读起来没劲
- 爽点出得太晚了
- 高潮完了没有余波
- 过渡章太水，像在赶路
- 看点不够密，读者容易滑走
- 帮我按番茄风格改单章
- 看看这一小段剧情该怎么重排

## 输入类型

- 单章正文
- 2 到 5 章连续正文
- 章节大纲 / 分章提纲
- 一段剧情链说明

如果用户给的是整本书设定，但没有给章节内容，先要求至少补一个具体章节、提纲或剧情链，否则只能给原则，不能做真正改稿。

## 工作模式

### 1. 先判断这次是哪个模式

- `诊断模式`：指出节奏、冲突、爽点问题
- `改稿模式`：直接重排或改写
- `重构模式`：重做章节结构、高潮链和反馈链
- `混合模式`：先诊断，再给一版改法

只要用户给了现成章节文本，默认走 `混合模式`。

### 2. 先抓主线，再抓三件事

在动刀前先回答三个问题：

1. 这一章的 `章节目标` 是什么
2. 这一章的 `核心冲突` 是什么
3. 这一章给读者的 `主要爽感回报` 是什么

如果这三个答案说不清，就不要急着润色，先修结构。

如果用户说这是“过渡章”，额外先判断一件事：

- 这章到底是在 `结算上一轮变化`
- 还是在 `铺下一轮期待`

如果两件事都没做到，这章大概率就是空转。

### 3. 三项检查

详细标准见 [docs/CHECKLIST.md](docs/CHECKLIST.md)。

#### 节奏

看四件事：

- 有没有无效铺垫
- 有没有偏离主线
- 场景切换是否自然
- 规则和能力是否前后一致
- 所谓过渡段是不是只剩移动、交代、聊天，没有新的期待

#### 冲突

看三件事：

- 阻碍够不够硬
- 高潮前有没有铺垫
- 高潮后有没有后果
- 平静段里有没有暗流、信息差、后续危机种子

#### 爽点

看四件事：

- 爽点类型是不是太单一
- 爽点是不是来得太突兀
- 爽点是不是拖太久不落地
- 爽点后有没有强反馈
- 看点密度是不是太稀
- 关键事件有没有 `铺垫 / 过程 / 余波`

### 4. 数据化改单章

如果用户提到番茄后台数据，按下面顺序处理：

1. 先看 `章节读完率` 掉在哪
2. 如果第 N 章掉得明显，优先从第 `N-1` 章开始查
3. 先定位具体问题：无聊、看不懂、拖、爽点没落地、钩子断了
4. 修改时保留关键剧情连续性，不为了一处掉点把整条主线掀翻
5. 区分 `小毒点删除` 和 `大结构返工`

## 输出方式

### 诊断模式

建议按下面顺序输出：

1. `一句话判断`
2. `这一章真正的问题不在表面，而在什么地方`
3. `节奏 / 冲突 / 爽点` 三项结论
4. `如果是过渡章，它是在结算什么、又在吊什么`
5. `优先修改顺序`
6. `必要时给重排方案`

推荐表格：

| 项目 | 结论 | 说明 |
|---|---|---|
| 节奏 | 通过 / 预警 / 命中 | 只写最关键问题 |
| 冲突 | 通过 / 预警 / 命中 | 只写最关键问题 |
| 爽点 | 通过 / 预警 / 命中 | 只写最关键问题 |

### 改稿模式

改稿时优先顺序固定：

1. 砍无效内容
2. 拉直章节目标
3. 补阻碍和对抗
4. 提前或重做爽点落地
5. 补高潮后的反馈
6. 最后再修措辞

如果是所谓“过渡章”，优先顺序改为：

1. 先删纯赶路内容
2. 补 `结算时刻`
3. 补下一轮期待
4. 再看是否需要小冲突或小钩子

### 重构模式

如果原章已经散掉，直接给出：

1. `原结构为什么散`
2. `新的章节推进链`
3. `新的高潮点`
4. `高潮后的反馈点`
5. `如有需要，附 1 段重写示例`

## 改稿原则

- 不迷信“多写就是充实”，无效内容要敢删
- 打斗、争吵、奔逃本身不是冲突，冲突是角色目标受阻
- 爽点不是一声吼出来的，它必须有压制、反击、落地、反馈
- 高潮不是孤岛，高潮后必须改动人物、关系、主线或局势
- 不存在纯粹负责赶路的“过渡章”，哪怕缓，也要让读者产生新的期待
- 如果是番茄导向，优先让读者“读得下去、想追下去、爽得出来”

## 章节重排常用公式

### 公式一：小冲突快落地

`被压 -> 反击 -> 反馈`

适合打脸、系统、逆袭、小高潮章。

### 公式二：递进冲突

`目标出现 -> 阻碍升级 -> 代价上升 -> 爆点兑现`

适合升级、闯关、比赛、谈判、追查。

### 公式三：高潮余波

`高潮爆发 -> 外部后果 -> 内部变化 -> 下个钩子`

适合大战、真相揭露、身份曝光、复仇成功后续。

### 公式四：拉新密度控制

`300字小看点 -> 500字小钩子 -> 1章1悬念`

适合拉新向、新媒体感、快节奏导向章节。

### 公式五：结算时刻

`事件结束 -> 情绪回弹 -> 关系/利益变化 -> 新风险冒头`

适合大战后、打脸后、晋级后、身份曝光后。

### 公式六：反流水账充实法

`铺垫 -> 过程 -> 余波`

任何核心事件如果只剩结果，没有过程和余波，就容易发空。
