# 番茄游戏题材

## 一、边界与易混淆

服务游戏文。适用：

- 虚拟网游
- 游戏异界
- 电竞 / 体育竞技
- 游戏直播

## 二、流派/分类选型

先判断是哪一种，边界处理不同：

1. **虚拟网游**：现实和游戏边界清晰
2. **游戏异界**：游戏世界就是生活世界
3. **竞技**：冠军、胜负、职业成长是主轴

## 三、核心爽点/抓手

游戏文的爽 = **清晰的成长反馈 + 一眼能看出独特性的竞争抓手**。

- 升级感要清楚：读者要能感到"变强了"，不只是等级数字变大。
- 抓手可以是金手指、游戏机制、战术风格、职业目标或团队关系，要让读者立刻看出和同类文的差别。

## 四、开篇切口

- 开头用**新奇事件**切入，别从注册账号、教程关卡这种平淡处起。
- 术语少而直观：一上来就术语爆炸，读者看不懂就走。

## 五、人设与金手指

- 游戏成长要和人物成长挂钩，别让主角在游戏里开挂、下线是个空壳。
- 核心抓手要尽快兑现出比别人强或不同的具体表现。

## 六、常见误区

1. 一上来术语爆炸，读者看不懂
2. 只有等级数字，没有实际爽感
3. 游戏成长和人物成长脱节
