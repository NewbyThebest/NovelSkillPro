# 番茄古言题材

## 一、边界与易混淆

只用于：

- 古代言情
- 架空古言
- 侯门 / 王府 / 朝堂恋爱
- 古风婚恋、权谋恋爱

易混淆：如果核心是宫斗宅斗算计、斗争是主菜，走 `gongdou-zhaidou.md`。古言的主菜是**情感线**。

## 二、流派/分类选型

古言先看四个底盘，缺了就立不住：

1. `身份差`
2. `礼法压制`
3. `关系束缚`
4. `情感推进空间`

古言不是换上古装词汇就行，必须让**身份和规矩真的能影响剧情**。

## 三、核心爽点/抓手

古言的味道来自**关系张力**，不是辞藻：

`处境压迫 -> 关系碰撞 -> 利益牵扯 -> 情感留口`

表面克制、内里拉扯，是古言的核心质感。

## 四、开篇切口

开书时优先定：

- 女主处境
- 男主权力 / 身份
- 二人第一次强关系接触点
- 第一阶段情感和利益冲突

稳妥起手不是大段背景，而是让处境压迫和关系碰撞尽快发生。

好用抓手：婚约 / 替嫁 / 被迫同盟、家族利益和个人情感冲突、礼法压制下的小动作和反制。

## 五、人设与金手指

古言多为现实向情感，主角的"牌"是身份、娘家势力、才学、对礼法的巧用，一般无强金手指（重生/穿越除外）。

男女主要**在关系和博弈里立人**，不靠旁白说"他们相爱了"。

## 六、常见误区

- 只有古风词，没有古代关系压力
- 角色说话太现代
- 情感线推进全靠口头心动（"她突然就爱上了"）
- 配角全是功能人，家族结构不成立
- 开篇先讲世界，不先讲处境

## 七、题材特有约束

- **身份和礼法必须真正参与剧情**：它们要成为情感的阻力和博弈的筹码，不是背景装饰。
- 情感推进要靠**具体事件和博弈**兑现，不靠心理独白宣布。

## 设定阶段必备维度

按顺序确认：

1. 时代、身份、家族结构与礼法边界，哪些规矩会真实限制人物。
2. 男女主的当前目标、身份差、关系起点和不能直接越过的阻力。
3. 家族、婚姻、朝堂或职业资源如何影响关系选择。
4. 情感推进依靠哪些具体事件、利益交换和信任变化。
5. 重生、穿越或其他特殊资源的来源、限制和不能替代的情感选择。
