# 模块：情绪调性与节奏控制

> 在第一步确定情绪定位、第四步设计章纲、第五步创作时加载。

## 情绪定位参数

| 参数 | 选项 |
|------|------|
| 虐心程度 | 轻虐 / 中虐 / 重虐 / 极虐 |
| 爽感程度 | 低爽 / 中爽 / 高爽 / 极爽 |
| 女主攻击性 | L0无法攻击 → L5暴力复仇 |

### 虐/爽矩阵（参考定位）

| | 低爽 | 中爽 | 高爽 | 极爽 |
|------|------|------|------|------|
| **轻虐** | 日常冷暴力 | 冷处理+新生 | — | — |
| **中虐** | — | 情感消耗 | 身份碾压 | — |
| **重虐** | 消失无音讯 | 失忆/被迫 | 重生/新伴侣 | — |
| **极虐** | 死亡/遗嘱 | 婴儿死亡 | 代理复仇 | 公开审判+碾压 |

## 爽点类型（按阶段）

| 阶段 | 爽点级别 | 适用类型 |
|------|---------|---------|
| 势能期 | 微爽 | 女主一句话噎住对立方/默默准备反击（读者知道角色不知道）/闺蜜帮腔 |
| 断裂期 | 中爽 | 做了以前绝不会做的事/签字/摘戒指/拉黑/"离婚吧" |
| 崩塌期 | 大爽 | 物证炸弹/男主跪地/对立方当众撕面具/全场反转 |
| 新生期 | 余韵爽 | 新日常的具体细节/旧世界废墟对比/她过得比他好 |

---

## 钩子类型

| 类型 | 示例 |
|------|------|
| 悬念钩 | "信封里还有第二页。她没有翻。" |
| 威胁钩 | "门外的脚步声停在了她房间前面。" |
| 反转钩 | "他打开文件，签名栏上已经有了另一个男人的名字。" |
| 揭秘钩 | "照片背面有一行字。她认出了那个笔迹。" |
| 失联钩 | "电话再也打不通了。" |

**禁止的章末写法**：
- ❌ "也许这就是命运吧。"
- ❌ "我终于明白了什么是真正的爱。"
- ❌ "那一夜，我哭了很久。"
- ❌ "就这样，一天过去了。"

## 对话节奏规则

| 阶段 | 女主 | 男主 | 对立方 |
|------|------|------|--------|
| 势能期 | 短回应："好""知道了""随你" | 命令/敷衍："乖""别闹""回头说" | 绵里藏针长句 |
| 断裂期 | 短、冷、终结："离婚吧""太迟了" | 突然词穷："……你说什么？" | — |
| 崩塌期 | 话越来越少→最终沉默 | 话越来越多→恳求→"回来""求你" | 歇斯底里暴露真面目 |
| 新生期 | 靠近时话多，被推开后骤减 | 温柔长句↔冷冽短句震荡 | — |

---

## 每章主打情绪选择

一章打透一个情绪。不要又虐又爽又恐惧——选一个，所有事件围绕它编排。

| 情绪 | 触发方式 | 适用阶段 |
|------|---------|---------|
| 愤怒 | 对比剪切/对立方推波助澜/男主自欺 | 势能期 |
| 心疼 | 女主的付出被无视/身体受伤 | 势能期 |
| 震惊 | 物证出现/身份揭露/真相翻转 | 任意 |
| 心碎 | 信息不对等（读者知道角色不知道）/遗物/录音 | 势能→崩塌 |
| 解气 | 爽点兑现/打脸/公开审判/沉默拒绝 | 断裂→崩塌 |
| 恐惧 | 对立方暴露犯罪性质/生命威胁 | 势能→断裂 |
| 释然 | 新生活细节/旧创伤放下的具体动作 | 新生期 |
