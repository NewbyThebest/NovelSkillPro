# 模块：结构模板

> 在第四步·大纲与章纲时加载。提供四阶段脊柱、逐章事件模板、开局写法。

## 四阶段脊柱

```
势能积累(25-40%) → 断裂/脱离(15-20%) → 崩塌(20-30%) → 新生/安息(15-25%)
```

### 阶段细节

**势能积累**：通过正在发生的具体事件展示付出和被忽视。每1-2章一个"微爆发未遂"——她几乎要反抗，但被一个具体原因拉回来（不是"她还爱他"，是"母亲的医药费下周到期"/"合同还有三个月"/"她摸了一下肚子"）。

**断裂/脱离**：核心是一个不可逆行动，不是一个决定。"她决定离开"不是断裂；"她签了字/拉黑了/嫁了别人"才是。

**崩塌**：通过物证链驱动。男主自欺六阶段：否认→自信→焦虑→愤怒→恐慌→崩溃。对立方在此阶段因自身缺陷自然暴露。

**新生/安息**：通过具体生活细节展示（新城市的公寓阳台浇花/教第一堂课/邻居端来砂锅），不是旁白"她终于自由了"。

## 逐章事件模板（短篇8-12章）

> **Ch1-3 = 开篇三章，付费转化关键区**（详见 §开篇三章）。Ch4 起按四阶段脊柱推进。

| 章 | 阶段 | 必须发生的事件 | 爽点 | 禁止 |
|----|------|-------------|------|------|
| 1 | 势能 | **正在发生的伤害**——开篇即事件。**必须使用场景OPEN-1**：3行引爆冲突，200字内读者站队 | 微爽：一个反常的冷静反应 | ❌ 概述背景/世界观铺垫/前300字主动交代前史 |
| 2 | 势能 | **第二次伤害，维度不同**。章头3行即新冲突，每200字一个刺痛点 | 微爽：暗中开始行动 | ❌ 重复同维度/平静日常开头/回顾上章 |
| 3 | 势能→裂 | **最后一根稻草**。本章末尾 = **全书最强悬念断点**——在最紧张处切断，驱动付费 | 中爽：做了以前绝不会做的动作 | ❌ 大段内心戏/章末情绪平复/阶段性收束 |
| 4 | 断裂 | **不可逆切割行动** | 中爽：签字/摘戒指/那个"咔哒" | ❌ 只决定不行动 |
| 5 | 崩塌 | **他发现第一个物证** | 解气爽：报应开始了 | ❌ 别人口述 |
| 6 | 崩塌 | **物证升级+对立方自爆** | 大爽：真面目/现实打脸 | ❌ 女主主动揭发 |
| 7 | 崩塌 | **挽回被拒**——沉默不用泪 | 大爽：沉默比台词有力 | ❌ 女主动摇 |
| 8+ | 新生 | **新旧世界具体对比** | 余韵爽：新日常vs废墟 | ❌ 旁白"终于自由" |

## 开篇三章（付费转化关键区）

**Ch1/Ch2/Ch3 是"投放素材"，不是"故事的开头"。** 每一行的唯一目的是让读者在 Ch3 末尾愿意付费。Ch3 末尾 = 付费转化断点，决定整部作品的商业生死。

第4章起按四阶段脊柱推进，不再受本节约束。

---

### 一、第1章：3行定生死

#### 开局五种强开法

| 类型 | 写法 | 示例 |
|------|------|------|
| 危机现场 | 第一句即紧急事件，身体痛感开场 | "剖腹产伤口还在渗血，他把离婚协议拍在了我的病床上。" |
| 当众羞辱 | 公开场合的不可逆剥夺 | "'今晚的Luna王冠给Elara。'他话音未落，她已经笑着伸出了手。我指上的戒指还热着。" |
| 反常行为 | 女主做了绝不会做的事（无解释） | "她把七年从未摘下的婚戒放在洗碗池边，拧开水龙头，看它被冲进下水道。" |
| 对比剪切 | 两个画面并置，一句完成 | "我在急诊室签病危通知书。他在隔壁包厢给另一个女人过生日。" |
| 物证炸弹 | 开篇即发现致命物证 | "衣柜最里面那件西装口袋里，掉出来一张收据。日期是我流产住院那天，消费地点是一家珠宝店。" |

#### 前200字节奏模板

```
[第1-3行 / 约50字] → 冲突引爆
  必须完成：一个可见的不公事件正在发生
  读者状态：愤怒/震惊/心疼——至少激活一个

[第4-8行 / 约50-80字] → 伤害加码 + 读者站队
  必须完成：第二层伤害落地（对立方亮相/旁人反应/更多细节）
  背景泄露方式：只通过对话碎片或动作细节，禁止叙述交代
  读者状态：明确知道"谁被欺负了"+"我讨厌谁"

[第9-15行 / 约80-100字] → 女主第一个反常反应
  必须完成：女主做了一个"不像她"的动作（读者好奇为什么）
  读者状态：好奇+期待——"她要干什么？"
```

#### 第1章·节奏硬指标
| 指标 | 要求 |
|------|------|
| 冲突密度 | 每100字至少一个刺痛点 |
| 对话间叙述 | 对话之间最多2句叙述 |
| 章末钩子 | 必须让读者"物理性放不下"——不是想知道后面，而是不看不行 |

---

### 二、第2章：持续加码，不给喘息

读者已上钩但还没付费——**核心是不让读者有任何"我待会儿再看"的念头**。

#### 第2章·章头规则

章头前3行必须有新的冲突/发现/危机。绝不能以缓冲方式开场。

| ✅ 允许的章头 | ❌ 禁止的章头 |
|-------------|-------------|
| 新打击落地："我推开门，他和她坐在我们的沙发上。她穿着我的睡衣。" | 回顾上章："昨晚的事让我一夜没合眼" |
| 上章钩子的残酷兑现："信封第二页只有一行字。是她的名字。" | 平静日常："第二天早上，阳光照进窗户" |
| 时间跳切到新危机："三天后。律师说他冻结了我的账户。" | 内心反思："我开始思考我和他之间到底出了什么问题" |
| 对话直接开场（高冲突）："你以为她真的爱你？"声音从我背后传来。 | 环境描写铺垫："夜色很深，街灯昏暗……" |

#### 第2章·节奏硬指标

| 指标 | 要求 |
|------|------|
| 冲突密度 | 每200字至少一个刺痛点 |
| 微反转 | 至少2个小意外（信息碎片/方向突变/对立方出招） |
| 背景交代 | 仍然只能"泄露"——单次≤1句，两次之间隔≥3行剧情推进 |
| 对话间叙述 | 对话之间最多3句叙述 |
| 章末钩子 | 必须让读者"物理性放不下"——不是想知道后面，而是不看不行 |

---

### 三、第3章：付费转化章 = 全书最强钩子

这一章的章末钩子**直接决定付费转化率**。是整部作品中商业价值最高的300字。

#### 第3章·结构要求

```
本章前80%：正常推进，但节奏持续紧张，不可松弛
本章最后20%：将核心冲突推到临界点——"即将爆发但还没爆发"
最后3行：在最紧张的点上切断。
```

#### 付费转化钩·四种收尾模式

| 模式 | 做法 | 示例 |
|------|------|------|
| 剪刀式 | 在动作进行到一半时切断 | "她举起手，对准了他的——" |
| 揭露式 | 揭露一半真相，留一半 | "信里第一行：'她不是你的女儿。'还有第二行。她没有看。" |
| 绝境式 | 把女主推到最危险的位置 | "身后是悬崖，面前是十二双红眼睛。她笑了。" |
| 反转式 | 推翻读者到目前为止相信的一件事 | "她扫了一眼那张照片。那里面的人……是她自己。" |

#### 第3章·禁止清单

- ❌ 任何形式的阶段性收束（"就这样，她做出了决定"）
- ❌ 任何情绪平复（章末不能让读者"放下心来"）
- ❌ 任何完整闭合的事件（核心事件必须在最紧张的点上被切断）
- ❌ 总结/感悟/释然（Ch3 最后一句是焦虑/震惊/恐惧，不是理解/成长）

### 四、开篇三章通用禁止清单

以下规则适用于 **Ch1/Ch2/Ch3**：

| 禁止行为 | 典型病句 | 问题 |
|----------|---------|------|
| 概述式开场 | "我叫Aria，我和Kael结婚七年了" | 零冲突，读者秒退 |
| 定语从句塞背景 | "作为陪伴他从荒野走到巅峰的女人" | 叙述者暂停剧情去解释 |
| 主动回忆前史 | "三年前他把她从废墟中救出来" | 冲突被暂停，节奏断裂 |
| 大段世界观铺垫 | "在这片由狼人统治的大陆上……" | 读者还没在乎角色就先要消化设定 |
| 对话间夹长描写 | 一句对话后接3-4句场景描写 | 对话节奏被拖慢到失去商业感 |
| 旁白式内心独白 | "我看着他，心想这五年到底值不值得" | 叙述者在分析，不是在经历 |
| 群演过多反应 | 连续写3-4个路人的反应/台词 | 稀释主线冲突的冲击力 |
| 连续背景叙述 | 超过2句的连续背景交代段落 | 信息灌注打断冲突节奏 |

### 五、改写示范（同一个开头的慢版 vs 快版）

**❌ 慢版**（典型AI生成节奏）：
```
"今晚的Luna王冠，我决定赐给Elara。"
Kael低沉浑厚的声音在满月祭坛上回荡。
他手里那顶本该属于我的骨月王冠，稳稳地落在了他身侧
那个穿着纯白长裙、瑟瑟发抖的女孩面前。那是他一个月前
在领地边缘"捡"回来的流浪Omega。
全场死寂。所有的目光像针一样扎向站在台阶下的我。作为
陪伴他从荒野废墟一路杀到今天巅峰位置的女人，所有人
都在等我这个被抛弃的"糟糠之妻"崩溃痛哭。
```
**问题**：第1行是爆点，第2行就退回去解释，第3-5行全是背景铺垫，到第7行还在交代前史。200字过去了，女主还没有任何行动。

**✅ 快版**（开篇三章节奏）：
```
"今晚的Luna王冠，给Elara。"
他把王冠放在那个女人面前的一瞬间，我无名指上的纽带戒
灼烧了一下。
Elara伸手去接。她的手在抖——但嘴角的弧度，一点都不抖。
"Aria姐姐别难过，"她回头看我，声音又轻又软，"Alpha说
你可以留下来当侍女。"
全场在等我哭。
我没哭。我低头看了一眼自己的手。这双手三天前刚为他熬
了一整夜修复防御结界，指尖的灼伤还没退。
我笑了一下。然后解开了领口那件被我洗到发白的旧披风。
```
**改善点**：❶ 前3行冲突落地+身体反应。❷ 对立方第4行就亮相且暴露真面目。❸ 背景通过"三天前熬了一整夜修复结界"用动作泄露，不是叙述。❹ 200字内女主已开始行动。

## 章纲格式（每章必填项）

```
第X章
- 起：[入口——承接上章什么/制造什么切入。Ch1-3：前3行必须是冲突/发现/危机]
- 承：[推进什么冲突]
- 转：[什么意外/反转——本章高潮]
- 合：[收束到什么新状态+章末钩子。Ch3 必须在最紧张处切断]
- 事件清单：[2-3个不可逆事件，标注维度]
- 对比剪切：[是否需要？A面/B面]
- 虐点：[触发方式：对话/行动/发现]
- 爽点：[类型+具体是什么]
- 主打情绪：[一个具体情绪]
- 钩子：[类型：悬念/反转/威胁/揭秘/失联 | 必须满足三重压力：强情绪+强冲突+悬念]
- 物证：[本章出现/被发现的物件]
- 对话节奏：[本章所处阶段]
- 性格驱动：[本章核心冲突触发了哪个角色的哪条性格烙印？→ 该角色基于小传会做出什么决断？→ 决断如何推动下一步剧情？]
```
