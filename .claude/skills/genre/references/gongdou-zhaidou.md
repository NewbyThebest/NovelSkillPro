# 番茄宫斗宅斗

## 一、边界与易混淆

只用于：

- 后宅争斗
- 宫廷妃嫔博弈
- 嫡庶、嫁娶、掌家、夺宠、站队
- 斗争本身就是主卖点

易混淆：如果核心是男女主情感线、斗争只是背景，走 `guyan.md`（古言）。斗争是主菜才留这里。

## 二、流派/分类选型

先把这四件事问清楚，否则斗会变成碎碎念和重复吵架：

1. 谁在争
2. 争什么
3. 输了会失去什么
4. 主角靠什么赢

四件事不清楚，先别急着堆坏人。

## 三、核心爽点/抓手

宫斗宅斗的爽不是单纯骂人，是**算计成功、局势反转、对方失势、主角上位**：

`利益受损 -> 表面压制 -> 暗手布局 -> 反击翻盘 -> 地位变化`

## 四、开篇切口

书名尽量写出：身份 + 斗争对象 + 反击钩子。

开篇优先抛：

- 被算计
- 被夺资源
- 被压制
- 被逼到必须出手

让读者一上来就看到主角的处境和"这口气得争回来"。

## 五、人设与金手指

宅斗主角的"牌"通常是：重生先知、掌家手腕、娘家势力、对手的把柄、隐藏身份。

主角要**会布局、能忍、该狠时狠**，不能只会当场对骂。配角（各方势力）要有层次，不是一个坏法。

## 六、常见误区

- 冲突太琐碎，不涉及核心利益（争来争去都是鸡毛蒜皮）
- 只会互呛，不会布局——斗成了吵架
- 主角反击太突兀，前面没有铺垫
- 配角全部一个坏法，没有层次
- 情绪很满，但斗争结果不改变局势——斗完地位没变，白斗

## 七、题材特有约束

- 每一场关键斗争都要**改变局势**（谁上位、谁失势、谁掌了权）；试探或铺垫场也至少要改变信息、站位或后续筹码，不能原地重复吵架。
- 反击要有**前铺的暗手**，不能天降翻盘。

## 设定阶段必备维度

按顺序确认：

1. 争斗场域、身份层级、可动用的名分与规则。
2. 各方争夺的具体资源，失败会失去什么，胜利会改变什么位置。
3. 核心人物的站队、底线、信息来源和可用筹码。
4. 试探、布局、反击和局势变化的因果链。
5. 斗争之外的情感、家族或朝堂关系如何反过来施压。
