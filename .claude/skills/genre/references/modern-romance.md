# 番茄现言题材

## 一、边界与易混淆

只用于：

- 现代言情
- 豪门、娱乐圈、校园、婚恋、真假千金
- 女主职业成长文

## 二、流派/分类选型

现言默认要看**双线**，最怕只剩恋爱没剧情、或只剩剧情没情感牵引：

1. `感情线`
2. `剧情线`

先定这本主打哪一条感情/剧情组合（豪门虐恋、职业成长+恋爱、真假千金+复仇等）。

## 三、核心爽点/抓手

现言的抓手在**成熟套路里的新切口** + **双线并行的情绪**：

- 在读者熟悉的情绪预期里，换视角、换身份、换关系角色做新意
- 感情线给甜/虐，剧情线给推进和目标，两条互相带动

## 四、开篇切口

找**人物命运开始转折的节点**做开篇：

- 被迫结婚
- 身份曝光
- 一夜情后续
- 进新行业
- 回归原家庭

从转折点切入，比从平淡日常起手更快抓人。

## 五、人设与金手指

现言一般无金手指，女主的"牌"是职业能力、身份反转、娘家/背景、清醒的头脑。

**女主职业成长**要落到实处：清晰行业 + 晋升路径 + 职业技能发挥场景 + 行业矛盾，不能只是披着职业皮谈恋爱。

## 六、常见误区

- 男女主刚见面就空心心动
- 只有甜虐，不见主线推进
- 女主职业只是标签，不参与剧情
- 套路都对，但切入点太旧

## 七、题材特有约束

- **双线都要活**：感情线和剧情线各自成立、互相带动，不能一条线彻底沦为背景。
- 职业文的行业必须**真参与剧情**，不是恋爱布景。

## 设定阶段必备维度

按顺序确认：

1. 感情线与剧情线的主次组合，以及两条线分别要兑现什么回报。
2. 男女主的关系起点、当前目标、关系阻力和不能越过的边界。
3. 家庭、职业、身份或社会环境如何制造现实压力和选择代价。
4. 关键配角、旧关系和利益关系如何推动或阻碍两条主线。
5. 身份反转、重生或其他特殊资源的真实性、限制和情感后果。
