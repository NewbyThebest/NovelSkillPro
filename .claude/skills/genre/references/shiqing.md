# 番茄世情爽文

## 一、边界与易混淆

只用于：

- 家庭关系矛盾
- 婚姻冲突
- 亲情压迫和反击
- 现实题材的强情绪打脸短长篇

易混淆：如果强情绪是为投放转化服务、核心在免费期和付费卡点，那是新媒体打法，两者可叠加，但世情底盘仍看这里。

## 二、流派/分类选型

世情爽文按矛盾来源大致分：

1. `家庭内部`：偏心、扶弟魔、重男轻女、婆媳
2. `婚姻`：出轨、算计、婆家压迫、离婚反击
3. `亲情压迫`：原生家庭吸血、道德绑架

先定矛盾主战场，再往下写，别几条线都想要、每条都不深。

## 三、核心爽点/抓手

世情的爽是**憋屈到极点后的反击兑现**：

`具体委屈 -> 核心利益受损 -> 情绪爆点 -> 反击 -> 更大后果`

爽点不是骂赢一架，是**主角把被剥削的东西夺回来、让剥削者付出代价、地位彻底反转**。

## 四、开篇切口

世情若使用**导语**，或正文以首段承担导语功能，应做到：

- **具象**：不要泛泛"她一直受委屈"，给一个能让人一下代入的具体场景（当众被塞账单、彩礼被弟弟花光、产房外丈夫在陪别人）
- **和主角核心利益有关**：委屈要扎在钱、房、孩子、名分这些实处
- **快速积累情绪**：前几段就让读者替主角憋屈

## 五、人设与金手指

世情多为现实题材，一般无金手指，主角的"牌"是：

- 重生/先知（知道对方底细）
- 攒下的证据
- 娘家或自己的隐藏能力/身份
- 对方的把柄

主角要有清晰边界：能判断什么值得忍、什么必须反击；反应要和核心利益、关系代价相称，不能长期只承受不行动。

## 六、常见误区

- 导语重复书名信息，等于浪费开篇
- 冲突和主角核心利益不相连，读者不心疼
- 极端转折（离婚、断亲）没有足够积累，显假——鸡毛蒜皮突然升级到断亲，读者觉得跳
- 打脸只停留在吵架层，没有实质的利益夺回和地位反转

## 七、题材特有约束

- 极端转折必须**逐级积累**：小委屈→核心利益受损→情绪爆点，每一步垫够了，最后的断亲/离婚才可信。
- 现实题材，别写成悬浮爽文——矛盾要贴真实家庭/婚姻逻辑。
