# 番茄悬疑题材

## 一、边界与易混淆

服务悬疑 / 推理 / 灵异悬疑。适用：

- 推理探案
- 灵异风水
- 规则悬疑
- 悬爱
- 短篇悬疑氛围强化

## 二、流派/分类选型

先判断是哪一路，两路侧重完全不同：

1. **灵异风水**：气氛、因果、信息差更重要
2. **推理探案**：线索、逻辑、真凶迷惑性更重要

## 三、核心爽点/抓手

悬疑的核心不是一堆怪事，而是一条完整的悬念链：

`未知 → 怀疑 → 线索 → 推进 → 解释或反转`

## 四、开篇切口

- **推理探案**：离奇开篇，先抛出一个抓人的案件/异常。
- **灵异风水**：用反差环境和信息不对称制造压迫氛围。

## 五、人设与金手指

- 灵异风水优先：氛围、信息不对称、反差环境、主角成长。
- 推理探案优先：线索前后呼应、真凶不能太早露相、结局意外且合理。

## 六、常见误区

- 只有气氛，没有谜题
- 只有谜题，没有情绪牵引
- 真相全靠作者硬揭
- 线索断裂，前文埋的东西后面忘了

## 七、题材特有约束

**合规与线索逻辑（悬疑必守）：**

- 不细写封建迷信宣传
- 不细写血腥和猎奇恶俗
- 不打擦边球
- 真实机关和现实政治要避开
- 线索要前后呼应闭环，埋了就要回收，别断裂

## 设定阶段必备维度

按顺序确认：

1. 核心谜面、异常或案件，以及读者第一阶段必须追问的问题。
2. 真相层级、线索分布、信息差和可验证的推理或灵异规则。
3. 受害者、嫌疑人、调查者和关键关系的目标与隐瞒动机。
4. 调查推进、风险升级、误导与反转的因果链。
5. 真相揭示、情绪回收、合规边界和不能靠作者硬揭的部分。
