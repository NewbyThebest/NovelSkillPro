# 番茄仙侠题材

## 一、边界与易混淆

适用：凡人流、古典仙侠、日常风仙侠、洪荒西游向、明确以修仙/宗门/资源争夺/求长生为主轴的作品。

- 传统玄幻 → `xuanhuan.md`
- 都市修真但核心是都市爽文 → `urban.md`
- 脑洞签到修仙、核心卖点是系统套路而非仙侠底盘 → 配合 `brainhole.md`

## 二、流派/分类选型

先定流派再动笔，流派没定就先别改文：

1. **凡人流**：主角资质普通，靠机缘、智商、苟发育慢慢崛起。
2. **古典仙侠**：门派、秘境、夺宝、儿女情仇、仙人江湖。
3. **日常风仙侠**：师门日常、轻松幽默、关系感强。
4. **洪荒/西游/大背景神话流**：先天自带大舞台和设定包袱。

## 三、核心爽点/抓手

底盘公式：**长生欲望 + 修炼体系 + 资源矛盾 + 个人成长。**

- 先有修炼目标，再有剧情。
- 先有资源争夺，再有冲突。
- 先有成长路径，再有爽点。
- 先让读者看见主角怎么活下去/变强，再慢慢铺大世界。仙侠底盘不是空讲仙气。

## 四、开篇切口

1. 凡人流少写凡间平淡日常，尽快切到**奇遇 + 入修行路**。
2. 开头要简单，不要先堆一排人物、宗门、设定词。
3. 情节优先，环境意境次之。
4. 先给读者一个明确期待：进宗门 / 抢机缘 / 脱离必死局 / 拿到金手指但要立刻变现。

## 五、人设与金手指

更稳的人设：聪明、果断、知道趋利避害、不轻易犯蠢。

凡人流常见安全做法：灵根差、出身低、金手指偏辅助、升级不飞天但能持续给回报。

金手指优先做成：加速修炼 / 提纯资源 / 推演功法 / 复制或培育灵药。少做一上来就无脑打脸。

## 六、常见误区

- 把仙侠写成玄幻打脸文。
- 开篇只顾氛围，不顾抓力。
- 主角不聪明，却硬写残酷修仙世界。
- 设定名词太多，读者进不去。
- 误以为仙侠一定要靠大脑洞，忽视情节和成长。
- 群像太早上量，新人自己先控不住。
