## 触发条件 (Trigger Scenarios)
当用户提出以下需求时，主动调用本技能：
- 要求“做个封面”、“生成书封提示词”。
- 要求“帮我设计一下封面”。
- 询问“我的小说适合什么样的封面”。

## 核心理念 (Core Concepts)
在辅助用户设计书封时，必须遵循以下原则：
1. **视觉传达故事**：封面是小说的第一张脸，必须精准传达小说的流派、基调和核心情绪。
2. **意象大于具象**：优秀的封面往往使用具有象征意义的核心意象（如破碎的王冠、孤独的背影），而不是像证件照一样直白地画出主角的脸。
3. **中文 Prompt 优先**：最终生成的 Prompt 必须是中文，以便用户可以直接理解和使用，或者在支持中文的国产 AI 绘图工具中使用。

## 上下文与前置条件 (Prerequisites)
在执行此技能前，Agent 必须：
1. **理解当前上下文**: 读取 `SOLOENT.md`（如果存在），提取小说的流派、核心主题、基调（如：清冷、热血、赛博朋克）以及主角的核心外貌特征或关键道具。
2. **收集必要信息**: 如果用户尚未提供，**必须主动询问**以下信息：
   - 书名（Title）
   - 作者名（Author Name）
   - （可选）一句话宣传语/Tagline

## 执行步骤与工作流 (Execution Steps)

### 第一步：提取与构思
- **动作**：结合 `SOLOENT.md` 中的信息，构思封面的核心视觉元素：
  - **背景 (Background)**: 符合世界观的场景（如：霓虹都市、缥缈仙山、奢华晚宴）。
  - **色调 (Color palette)**: 匹配故事基调的色彩组合（如：黑金对比、清冷蓝白、暗黑红黑）。
  - **核心意象 (Imagery)**: 能够代表故事核心的物品或人物剪影（如：破碎的王冠、交握的手、孤独的背影）。

### 第二步：填充标准模板
- **动作**：将构思好的元素和用户提供的书名/作者名填入标准的中文提示词模板中。

### 第三步：输出结果
- **动作**：向用户展示设计思路，并提供可以直接复制使用的中文绘图提示词。
- **输出格式**：请使用以下 Markdown 格式向用户展示结果：

```markdown
基于《[书名]》的设定（[简述提取到的流派和基调，如：清冷风现代言情]），我为您构思了以下书封设计方案：

### 🎨 视觉设计思路
- **核心意象**：[描述画面主体，如：维港夜色下一个孤独而决绝的女性背影]
- **色彩基调**：[如：深海蓝与暗金色的碰撞，体现克制与奢华]
- **排版建议**：[简述书名和作者名的摆放位置]

### 🪄 AI 绘图提示词 (Prompt)
您可以直接复制以下提示词到 Midjourney 或其他 AI 绘图工具中生成底图：

```text
一本名为《[书名]》，作者是[作者名]的小说专业封面设计。

尺寸：600x800 像素。
背景：[填入背景描述，如：模糊奢华的香港夜景，带有霓虹灯光]。
色调：[填入色调，如：深海蓝与暗金色的碰撞]。
核心意象：[填入核心意象，如：一个优雅女性的背影剪影，她看向远方，手里拿着一个破碎的指南针]。

排版布局：
[顶部 20%] 留白，带有[如：微妙的极光效果]
[中间 40%] 粗体、优雅的衬线字体，带有轻微的做旧纹理，用于书名《[书名]》
[底部 30%] 干净、精致的无衬线字体，用于作者名“[作者名]”
[底部 10%] 小字体的宣传语：“[宣传语，如果没有则留空]”

风格：电影级光影，极高细节，8k 分辨率，ArtStation 流行趋势，专业平面设计。
```

**💡 下一步**：您可以拿着生成的底图，用 Canva 或 Photoshop 将文字排版上去。如果对画面元素不满意，我们可以调整意象和色调！
```

## 自查清单 (Validation Checklist)
在输出书封设计前，进行以下自查：
- [ ] 是否已经读取了 `SOLOENT.md` 以获取小说的真实基调？
- [ ] 是否已经向用户确认了“书名”和“作者名”？
- [ ] 提供的 Prompt 是否为全中文？
- [ ] 视觉意象是否符合小说的核心流派（没有给古言配赛博朋克背景）？
