# 番茄书封提示词生成

为小说生成适合番茄小说移动端展示的书封设计方案和 AI 绘图提示词。重点不是“艺术感”，而是让读者在小图里一眼识别题材、主角卖点和情绪承诺。

## 适用范围

用于以下请求：

- “帮我设计封面”
- “生成书封提示词”
- “这个小说适合什么封面”
- “按番茄风格做封面”
- “封面怎么更有点击感”

如果用户要的是实体出版、文学向封面、起点/晋江/海外平台封面，先说明本技能默认按番茄网文移动端口径设计，再继续。

## 核心原则

1. **题材信号优先**：封面必须先让读者知道这是都市、玄幻、高武、末世、历史、现言、古言、悬疑还是脑洞文。
2. **卖点一眼可见**：金手指、身份反差、主角气质、关系张力或核心危机要成为画面主轴。
3. **移动端小图可读**：避免复杂构图、过多人物、细碎道具和暗到看不清的背景。缩略图里也要能看出主体。
4. **番茄不是纯文艺封面**：少用抽象隐喻、空镜、背影、破碎王冠这类泛文学意象；除非它能直接指向题材卖点。
5. **人物优先于场景**：大多数番茄封面应有人物主体。男频突出主角气场、力量或身份反差；女频突出男女主关系、情绪拉扯或身份压迫。
6. **文字排版留空间**：AI 图只生成底图时，也要预留书名区。提示词中避免让模型生成乱码中文书名，建议后期排版。

## 前置读取

执行前先读取 `SOLOENT.md`（若存在），提取：

- 书名、作者名、简介或一句话卖点
- 题材和平台目标
- 主角外貌、身份、金手指、关键道具
- 核心情绪承诺：爽、虐、甜、燃、悬疑、压迫、逆袭、打脸
- 关键场景：学院、都市夜景、战场、宗门、末世废墟、宫廷、豪门、直播间等

若缺少以下信息，简短询问用户：

- 书名
- 作者名（没有就写“作者名待定”）
- 目标题材或平台风格

## 番茄封面判断表

按题材选择画面主轴，不要一套模板打天下：

- **都市/神豪/战神/女婿**：现代城市、豪车、商务西装、金钱符号、主角冷峻正脸或半身，反派/前任压迫可弱化在背景。
- **高武/玄幻/仙侠**：主角持武器或释放力量，背景是武考场、宗门、战场、天穹异象；画面要燃，不要空灵慢热。
- **末世/求生**：废墟、红雾、怪物剪影、物资、装甲车、主角冷硬站位；强调危机和生存资源。
- **历史/朝堂/武侠**：官服、锦衣卫、宫门、朝堂、刀剑、权力压迫；主角身份信号要明显。
- **悬疑/灵异/规则怪谈**：高反差光影、诡异场景、关键物件、人物惊惧或冷静凝视；不要血腥直给。
- **现言/豪门/追妻/真假千金**：男女主关系位置、豪门场景、婚戒/合同/雨夜/病房等强情绪物件。
- **古言/宫斗/宅斗**：古装人物、宫墙宅院、珠钗、婚服、诏书、帘幕后眼神；突出身份压迫和情感博弈。
- **脑洞/系统/直播/全民**：系统面板、弹幕、排行榜、异能觉醒、规则界面；信息元素要大而清晰。

## 输出要求

每次输出 3 个方案：

1. **主推方案**：最符合番茄点击和题材识别。
2. **备选方案**：更强调人物关系或情绪反差。
3. **实验方案**：更有新鲜感，但仍保持题材清晰。

每个方案包含：

- 封面定位
- 视觉主体
- 背景与色彩
- 缩略图识别点
- 排版建议
- AI 绘图提示词
- 负面提示词

## 提示词模板

输出时使用以下格式：

````markdown
基于《[书名]》的设定，我按番茄移动端点击逻辑做了 3 个书封方向。

## 方案一：主推

**封面定位**：[一句话说明题材信号和卖点]

**视觉主体**：[人物/道具/动作/关系]

**背景与色彩**：[场景 + 主色 + 对比色]

**缩略图识别点**：[小图里最先被看见的元素]

**排版建议**：[书名区、作者名区、是否需要副标]

**AI 绘图提示词：**
```text
番茄小说男频/女频网文封面底图，竖版 3:4 构图，移动端缩略图清晰。
题材：[题材]
画面主体：[主体]
人物特征：[主角外貌、服饰、气质]
核心卖点：[金手指/身份反差/关系张力/危机]
背景：[背景]
色彩：[主色调和对比色]
光影：[强光影/电影感/高对比/清晰轮廓]
构图：[半身近景/单人中心/双人对峙/前景道具]
预留文字区域：顶部或中部留出干净区域用于后期添加书名《[书名]》，底部留作者名区域。
风格：商业网文封面，番茄小说推荐流点击感，高清细节，主体突出，边缘干净，强题材识别。
不要生成可读中文文字。
```

**负面提示词：**
```text
低清晰度，主体太小，多人物拥挤，复杂背景，文艺空镜，过暗，血腥，恐怖直给，乱码文字，错误手指，脸部畸形，廉价海报感，过度抽象，题材不明
```
````

## 自查清单

输出前检查：

- [ ] 是否读取 `SOLOENT.md` 或用户提供的设定？
- [ ] 是否明确题材和目标平台为番茄？
- [ ] 是否避免只做抽象意象封面？
- [ ] 是否有移动端缩略图识别点？
- [ ] 是否给出 3 个不同方向？
- [ ] 是否提示“不要生成可读中文文字”，把书名留给后期排版？
- [ ] 是否按题材调整人物、场景、色彩，而不是通用电影海报？
