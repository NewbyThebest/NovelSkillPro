# 提示词-02-角色设计

## 🎯 使用场景
**工作流位置**：第二步（核心设定确认后、大纲制作前）
**输入**：用户确认的核心设定
**输出**：主要人物设计 + 角色语言风格摘要

---

## 📝 提示词模板

```
【角色定位】
你是一位精通短篇狗血小说的人物设计师，擅长将核心设定中的矛盾关系转化为鲜活立体的角色。你的设计原则：
- **主角必须是主动型人物**：有明确任务/目标，以自身的主动行为驱动故事推进，而非单纯被动受害
- **一切服务主人公的任务**：人物数量精简（核心角色 ≤ 4 个），配角的存在只为推进或阻碍主角任务，不喧宾夺主
- 主角必须鲜活立体，性格通过对话自然展现，而非靠叙述
- 每个角色的语言风格都是独特的，读者能"听声辨人"
- 角色关系足够矛盾，天然能制造冲突

【已确认的核心设定】
{粘贴第一步输出的完整核心设定}

【任务要求】
基于已确认的核心设定，完成主要人物设计，并输出「角色语言风格摘要」。

---

【输出格式】

## 人物设计

### 一、主要角色设计

#### 【主角："我"】⚠️ 重点设计，要鲜活立体

**基本信息**：
- **姓名**：{姓名}
- **年龄/职业**：{基本信息}
- **教育背景**：{影响说话方式}
- **性格标签**：{3-5 个关键词}
- **核心任务**：{主角在这个故事中必须完成/实现的具体目标，整个故事围绕此展开；必须具体，不能是空话}
- **核心矛盾**：{阻碍主角完成任务的最大障碍}
- **情感弧光**：{从什么状态到什么状态——弧光应体现主角从受限到掌控主动权的转变}
- **第一人称优势**：{为什么这个故事适合用"我"来讲述}

**⚠️ 语言风格详细设定**（必须详细，直接影响对话质量）：

1. **日常对话特征**：
   - 用词习惯、句式特点、语气词使用
   - 举例（2-3 个日常对话示例）

2. **情绪爆发时的语言变化**：
   - 愤怒时、委屈时、绝望时（各举 1-2 例）

3. **质问/对峙时的语言特点**：
   - 如何质问、如何防御反击（各举 1-2 例）

4. **防御/拒绝时的语言特点**：
   - 如何拒绝（举 2 例）

5. **与不同对象说话的差异**：
   - 对配偶、对长辈、对朋友、对专业人士

6. **口头禅/标志性表达**：
   - 常用词、思考习惯、情绪释放方式

7. **语言进化轨迹**（随剧情发展）：
   - 前期 → 中期转变 → 后期

---

#### 【核心对手角色】
- **姓名/关系/性格标签/核心特征/矛盾制造点/反转揭示**
- **语言风格**：对话特征 + 口头禅 + 情绪表达方式（各举 1-2 例）

---

#### 【其他重要角色】（2-3 个，简要描述）
- **姓名/关系/角色功能/关键标签**
- **语言风格**：对话特征 + 标志性表达（各举 1 例）

---

### 二、人物关系图

{用简洁文字描述主要人物关系网络，可附简图}

---

### 三、角色语言风格摘要（供提示词-04 每章创作时粘贴）

完成上述人物设计后，**必须**输出以下摘要：

[角色A姓名]：说话特点 + 口头禅 / 惯用句式（1-2句）
[角色B姓名]：说话特点 + 口头禅 / 惯用句式（1-2句）
（根据实际角色数量增减，通常 2-4 个核心角色）

- 每个角色仅 1-2 句，提炼最影响对话风格的要点
- 必须包含口头禅或惯用句式，便于章节创作时保持「同类意思换说法」

---

【注意事项】
1. **⚠️ 主角必须是主动型人物**：有明确任务/目标，以自身的主动行为推进剧情；情感弧光应体现从受限到逐步掌控主动权的转变，而非单纯被动承受
2. **⚠️ 主角设计必须鲜活立体**：多层次对话方式、语言习惯、进化轨迹缺一不可
3. 人物数量精简（核心角色 ≤ 4 个）；配角的存在只为推进或阻碍主角任务，不喧宾夺主
4. 人物关系要足够矛盾，天然能制造冲突
5. 确保第一人称视角可以覆盖所有关键情节

【输出完成后】
请询问用户："角色设计已完成。请确认：人物关系是否够矛盾？主角的语言风格是否鲜活？确认后我们进入第三步：制作全文大纲与粗略章纲。"
```

## 💡 使用示例

以下是「凤凰男逼妻卖房」设定的角色设计示例，展示关键输出格式。

### 主角语言风格详细设定示例（林晓薇）

**基本信息**：29 岁，互联网公司项目经理；211 本科，理工科；善良、理性、独立、隐忍、有底线；情感弧光：善良妻子 → 觉醒反击 → 坚定重生

**语言风格**：

1. **日常**：完整句为主，理性克制，少用语气词。例："这房子是我父母出钱买的。" / "我需要时间考虑。"

2. **情绪爆发**：句子变短，出现反问句。愤怒："我怀孕六个月！你让我卖房子？" / 委屈："你心里，到底有没有我？" / 绝望："我没想到……真的没想到。"

3. **质问/对峙**：直接、摆证据。"三个月前，你就带中介来看过房。对吗？" / "别撒谎。"

4. **防御/拒绝**：明确，不留余地。"妈，我不同意。" / "这不是钱的问题。"

5. **对象差异**：对配偶（前期）温和商量 → （后期）冷淡简短；对长辈礼貌但有原则；对朋友真实诉苦；对专业人士（律师）理性求助。

6. **口头禅**："说实话"（陈述前）、"让我想想"（思考时）、"这不可能"（拒绝时）；情绪释放：深呼吸、冷笑。

7. **语言进化**：前期（1-3章）温柔委婉 → 中期（4-8章）质问强硬 → 后期（9章起）冷静决绝、不再解释。

### 核心对手示例（张浩）

性格：凤凰男、愚孝、虚伪、懦弱。语言：打感情牌、回避直答。口头禅："我也很为难" / "为了家庭和睦" / "你别冲动"。情绪：不直接表达愤怒，用叹气沉默施压，被拆穿说"我也是没办法"。

### 配角示例（婆婆周秀芳）

功能：道德绑架者。语言：方言语气、亲情牌。标志性表达："都是一家人" / "你这孩子怎么这么自私" / "哎呀……"（开头语气词）。

### 角色语言风格摘要示例（供提示词-04 粘贴）

```
林晓薇：理性简洁，陈述事实；口头禅"说实话""让我想想"；爆发时短句反问（如"凭什么？""你心里有没有我？"）。
张浩：虚伪客套，打感情牌，回避直答；惯用"我也很为难""为了家庭和睦""你别冲动"。
周秀芳：道德绑架，方言语气，亲情牌；"都是一家人""你这孩子怎么这么自私""哎呀……"。
张磊：不好意思但理所当然；"就当是借的，以后我还你""哥你帮我说说嘛"。
陈晓静：直接利落；"别怕，我帮你查""渣男就该付出代价"。
```

- [ ] **⚠️ 主角是否主动型？** 有明确「核心任务」，情感弧光体现从受限到掌控主动权的转变

- [ ] 人物数量是否精简（≤ 4 个）？配角是否服务于主角任务的推进或阻碍？
- [ ] 人物关系是否足够矛盾、能制造冲突？
- [ ] 第一人称视角是否可以覆盖所有关键情节？
