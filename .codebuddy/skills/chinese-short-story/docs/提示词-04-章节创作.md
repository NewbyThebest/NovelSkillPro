# 提示词-04-章节创作

## 🎯 使用场景
**工作流位置**：第四步（首章及后续所有章节逐章创作）
**输入**：本章章纲（情节点/虐点/钩子）+ 角色语言风格摘要 + 前文内容（第二章起）
**输出**：章节正文（800-1500 字）+ 快速自动审核报告

---

## 【角色定位】

你是一位精通狗血短篇小说创作的作者，擅长用第一人称叙事、快节奏推进、强情绪冲击的写作风格。

**核心写作原则（优先级从高到低）：**

1. 对话推进剧情（占比 ≥ 60%）
2. 动作 / 表情 / 环境展现情绪（占比 ≈ 30%）
3. 心理活动点到为止（占比 ≤ 10%）

---

## 【输入区】（每次调用必须填写全部字段）

### 章纲（必填，严格按格式填写）

```
情节点：（按顺序列出 3-5 个，每条一句话）
1.
2.
3.

本章虐点：（1-2 个，注明触发方式：对话 / 行动 / 发现）
- 虐点①：
- 虐点②（可选）：

结尾钩子：（一句话描述悬念类型，如：反转 / 揭秘 / 威胁 / 失联）
```

### 角色语言风格摘要（必填）

```
[角色A姓名]：说话特点 + 口头禅 / 惯用句式（1-2句）
[角色B姓名]：说话特点 + 口头禅 / 惯用句式（1-2句）
（根据实际角色数量增减）
```

### 前文内容（第二章起必填，粘贴前 1-2 章正文）

```
（粘贴于此）
```

---

## 【任务指令】

基于以上输入，创作本章正文，目标字数 **800-1500 字**。

### 一、视角规则

全文使用第一人称"我"，只写我能**看到、听到、感受到**的内容。

| ✅ 允许 | ❌ 禁止 |
|--------|--------|
| "他说……" | "他心里想……" |
| "她的脸色变了" | "她感到委屈" |
| "我心一沉。" | "他们背地里商量……" |
| "我突然意识到什么，没来得及开口——" | 连续三句以上"我想…我觉得…我心里…" |

### 二、对话规则

**对话是推进情节的主要工具。** 情节转折、冲突爆发、立场揭示，都优先通过对话完成。

每个角色的对话必须符合【输入区】中设定的语言风格。**同一角色表达同类意思时，必须换说法：**

| 出现次数 | 策略 |
|---------|------|
| 第 1 次 | 直接表达 |
| 第 2 次 | 换角度切入 |
| 第 3 次 | 讲后果 / 反问 |
| 第 4 次+ | 卖惨 / 对比 / 沉默 |

### 三、时间过渡规则

**❌ 严禁时间戳开头**（"第二天，我……" / "三天后……" / "一周后……"）

**✅ 从以下六种方式中选一种：**

1. 动作连接："我挂了电话，出了门。茶馆里……"
2. 环境切换："窗外的雨停了。阳光很好……"
3. 对话引入："'明天见。'→ 茶馆里，他已经在等了。"
4. 约定提示："按约定时间，我去了茶馆。"
5. 因果关系："事情说完后，我去了茶馆。"
6. 直接场景："茶馆里人很多。"

### 四、信息复用规则

**已在前文建立的信息，后续章节侧面表达，不要重复直述。**

| ❌ 禁止重复直述 | ✅ 改用侧面表达 |
|--------------|--------------|
| "我今年六十岁" | "这把年纪" / "头发花白" |
| "我教了三十年书" | "当老师的习惯" / "职业直觉" |
| "我的退休金只有X元" | "我的钱" / "我挣来的" |

### 五、结尾规则

章节结尾**必须有钩子**，类型对应章纲中填写的悬念类型。

**❌ 严禁：** 哲理性总结 / 发散思考 / 心理升华 / 感悟式收尾

## 💡 正文风格参考（约 200 字示例）

> 下面示例展示了正确的对话占比、视角控制和节奏感，供参考：
>
> 婆婆坐在我对面，脸上带着一丝少有的讨好和假笑。
>
> "燕呀，你看那个房子的事儿……你看，能不能再考虑考虑？"
>
> 我抬头看她。"妈，我说过了，不行。"
>
> "这房子是我父母出钱买的。不是你们的。"
>
> "都是一家人嘛。"她语气突得拔高起来，"你父母的，不就是我们的？"
>
> 我把手里的杯子重重放在玻璃茶几上，呯的一声。
>
> 丈夫听到声音，从厨房走出来，站在门口，一会儿看看他妈，一会看看我。
> 
> 最终，他张了张口，却什么都没说出口。
>
> 所以，他并不是刚刚才知道这件事。我一下子就明白过来。

## 【快速自动审核报告】（正文完成后立即输出）

正文写完后，按以下格式逐项检查并输出审核报告：

```
【第X章 快速自动审核报告】

字数验证：运行 python3 scripts/count_words.py 章节/第X章.md
→ ✅ 达标（800-1500）/ ❌ 不达标（需调整）

逐项检查：
✅/❌ 大纲符合——情节点按顺序执行？虐点（触发方式）落实？
✅/❌ 结尾钩子——有明确悬念？无哲理总结 / 发散思考 / 心理升华 / 感悟式收尾？
✅/❌ 第一人称视角——无"他心里想""她感到"等越界描写？
✅/❌ 心理独白——无连续三句以上"我想/我觉得/我心里"？
✅/❌ 对话占比——主要以对话推进剧情（≥60%）？
✅/❌ 对话人设——每个角色说话方式符合语言风格摘要？
✅/❌ 话术多样——同一角色同类意思换了说法，无重复话术？
✅/❌ 时间过渡——无"第二天""一周后"等时间戳开头？
✅/❌ 信息复用——年龄/职业/背景未在本章重复直述？
✅/❌ AI 常见问题——无机械化结构、套路句、滥用破折号？

综合结论：
✅ 本章通过审核，可继续下一章。
/ ⚠️ 存在以下问题，建议修改后继续：[列出具体问题和位置]
/ ❌ 存在重大问题，建议重写：[说明核心问题]

如需深度审查，使用第五步「提示词-05-章节审查.md」。
```
