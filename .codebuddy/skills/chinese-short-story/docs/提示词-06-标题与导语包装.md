# 提示词-06-标题与导语包装

用于为番茄短故事生成标题、导语和点击包装。可在开书前使用，也可在全文完成后精修。

## 输入

```text
书稿/大纲/核心设定：

主角与核心矛盾：

主要反转：

结局情绪：
```

## 标题原则

番茄短故事标题要在小图和列表里快速传达：

- 谁被伤害/欺骗/压迫
- 谁反击/重生/醒悟/揭穿
- 具体数字、身份、关系、物件
- 反差、悬念或打脸

少用抽象文艺标题，多用具象冲突标题。

## 输出格式

### 一、标题方案

给出 5 个标题：

1. **主推标题**：最强点击。
2. **冲突标题**：矛盾最直给。
3. **悬念标题**：最让人想知道真相。
4. **爽点标题**：突出反击/打脸。
5. **情绪标题**：突出心疼、愤怒或遗憾。

每个标题后说明：

- **核心钩子**：
- **适配读者情绪**：
- **风险**：是否太长、太剧透、太平、太像旧梗。

### 二、导语方案

给出 3 个导语，每个 80-150 字：

- **强冲突版**
- **悬念反转版**
- **情绪代入版**

导语必须做到：

- 第一行就有事件或结果
- 不剧透最终结局
- 点出人物关系和核心矛盾
- 结尾留下悬念或反击预期

### 三、主推组合

输出：

```text
主推标题：
主推导语：
推荐理由：
```

### 四、可选副标/短句

给 3 条可用于封面或简介的短句，每条 10-20 字。

## 示例方向

不要机械套用，但可参考这些结构：

- “老公让我净身出户，我反手晒出一份鉴定”
- “婆婆抢我拆迁款那天，我收到亲子鉴定”
- “闺蜜替我嫁豪门后，才知道我早就重生了”
- “女儿病危他陪白月光，我签下离婚协议”
- “全家逼我退学打工，录取通知书寄到村委会”

## 完成后询问

```text
我建议用第 X 个标题搭配第 X 个导语。你要走更狠的冲突版，还是更悬疑的反转版？
```

## 检查清单

- [ ] 标题是否具象，有人物关系和冲突？
- [ ] 是否避免文艺空泛？
- [ ] 导语是否 80-150 字？
- [ ] 第一行是否抓人？
- [ ] 是否没有剧透最终结局？
- [ ] 是否留下继续阅读的理由？
