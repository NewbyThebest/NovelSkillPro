# 战斗类型模板

按战斗类型选模板。一个场景可以叠加两种类型，例如“规则擂台 + 藏拙”“追杀 + 反杀”“越级战 + 强敌试探”。

## 规则擂台

适用：比武、训练考核、擂台赛、积分战、陪练测试、宗门试炼。

关键要素：

- 规则必须变成压力：倒计时、扣分、越线、禁止逃跑、必须反击、不能杀人、限定武器。
- 裁判/系统/教官负责结算爽点，不一定需要围观者。
- 对手可以利用规则逼主角犯错。
- 主角可以钻规则缝隙完成反击、藏拙、控分或保命。
- **切磋收力变体**：无杀意的比试、点到为止的师门过招——张力不来自生死，来自输赢面子、评价、赌注、暗中较劲。招要收，但要让读者看出双方都留了手、谁略胜一筹。别用生死战的力度写切磋。

节拍：

```text
规则公布或暗示 -> 对手用规则压人 -> 主角被迫暴露一部分能力 -> 反击得分/避开惩罚 -> 对手升级 -> 主角在规则边缘完成目标 -> 系统或裁判结算。
```

## 追杀反杀

适用：巷战、山林逃亡、夜袭、逃课式试炼、被围堵。

关键要素：

- 路线比招式重要：门、窗、墙、楼梯、树、河、废墟、盲角。
- 呼吸和体力是倒计时。
- 反杀不要突然，应提前埋可利用的地形、道具或误导。

节拍：

```text
危险预警 -> 被迫逃/边打边退 -> 伤势和体力恶化 -> 利用地形制造错判 -> 对手冲过头 -> 主角反杀或脱身。
```

## 群战

适用：一对多、混战、护送、突围、守门、救人。

关键要素：

- 不要同时细写所有人，永远锁定主角视线中的一两个威胁。
- 用“方位 + 距离 + 优先级”写清局面：左侧刀、背后箭、正面盾、脚下阵。
- 群战爽点来自调度：借敌挡敌、打乱阵型、擒贼先擒王、制造缺口。

节拍：

```text
包围成形 -> 主角识别最大威胁 -> 先破一角 -> 敌人补位 -> 主角借位/借尸/借兵器扩大混乱 -> 打开出口或斩首。
```

## 越级战

适用：主角弱打强、境界差、资源差、重伤硬撑。**前提是主角真的弱**。（和"强敌试探"的区别：越级战最终要拿下一个局部胜利；强敌试探最终认输、只求活着带走情报。打不过但要赢一点，用这个。）

关键要素：

- 强者必须让读者感到压迫：一招就逼主角付代价。
- 主角赢不能靠对手降智。真弱者赢，有两种正当赢法，可叠加：
  - **智取**：靠情报、地形、对手弱点、规则缝隙，用脑子换一个赢的机会。
  - **隐瞒底牌 / 绝境爆发**：藏一张牌，绝境时引爆——但底牌必须有代价或限制（反噬、时限、驾驭不了的风险），不能白捡变强通吃。
- ⚠️ **红线：赢得不能太顺。** 哪怕掏了底牌、爆了种，也要打得依然吃力、赢得依然狼狈。爆种后无伤平推、把强敌轻松打昏，越级的紧张感就塌了。真正的爽点是"赢了，但差点没扛住"，不是"我其实一直能赢"。胜利最好是局部胜利：撑过一招、伤到对手、夺到物品、拖到时间、保住人；就算完胜，过程也得够险、代价够重。
- **注：如果主角其实不弱、只是装弱最后掀桌碾压，那是"扮猪吃虎"，不属于越级战**——按"藏拙"（hidden-strength.md）+ "秒杀/打脸"（payoff-types.md）来写，爽点在揭穿反差和强者破防，而不是险死还生。

节拍：

```text
强者压场 -> 主角第一次吃亏 -> 发现规律/弱点 -> 用代价换机会 -> 短暂反制 -> 目标达成但留下后患。
```

## 秒杀战

适用：立威、打脸、清杂兵、展现实力差。

关键要素：

- 秒杀前要铺垫对方嚣张、身份、战绩、旁人预期。
- 出手过程要短，结算要足。
- 爽点在“别人以为会怎样”和“结果突然反过来”。

节拍：

```text
对手放狠话 -> 众人默认主角会输/麻烦 -> 主角只动一次 -> 对手倒下 -> 结算身份差和后果。
```

## 斗法/异能战

适用：仙侠斗法、玄幻术法、异能、武核、阵法、法宝。

关键要素：

- 先定规则：能力代价、范围、触发条件、克制关系、冷却或消耗。立了就要在打斗里用上，设了从不触发、对手从不利用，等于没设。
- 斗法不是光污染，要写“能力如何改变局面”。
- 用可见结果承载抽象力量：符纸燃尽、阵纹开裂、空气扭曲、地面结霜、金属失重。

节拍：

```text
能力亮相 -> 主角判断规则 -> 第一次应对失败或付代价 -> 找到克制/空隙 -> 对手加码 -> 主角用规则反制。
```

- ⚠️ **红线：主角必须用能力主动反制、至少作用到对手身上一次（打乱、压制、伤到、破防、反打其中之一）。** 全程只用能力躲、闪、跑，一次都没主动打回去，那不是斗法，是脱身戏——脱身戏可以写，但别当斗法交差，斗法的爽点是“用能力赢对手”，不是“用能力躲开对手”。自查：数一数主角的能力有几次用来攻击/反制对手，零次就是没打起来。

## 强敌试探

适用：主角暂时不能赢、boss初见、导师压迫、强敌测验。（和"越级战"的区别：这里最终认输，赢的不是战斗，是情报、时间、秘密或差距认知。打不过也不硬求胜，用这个。）

关键要素：

- 目标不是胜利，而是获得情报、撑住时间、保护秘密、确认差距。
- 结尾可以输，但要让主角拿到下一阶段的钥匙。

节拍：

```text
强敌随手压制 -> 主角意识到差距 -> 用小技巧换一次喘息 -> 强敌认真一瞬 -> 主角受伤但读出信息 -> 为下一次战斗埋钩子。
```

## 热兵器/枪战

适用：都市特勤、军警、谍战、末世火并、废土、丧尸围城、武者对火器。

关键要素：

- 枪战的压力不在互相扫射，在"暴露就死"：谁先动、谁露头、谁的弹匣空了。
- 掩体和距离是核心：墙角、车体、门框、货架，打光一段就要换位，钉在原地=挨枪。
- 用具体故障和动作制造节奏：点射、压制、换弹、卡壳、跳弹、哑火、听声辨位。
- 武者对火器要写清"快到子弹追不上"或"硬抗有代价"，别让主角站着挨枪还没事。
- 结果要具体：血雾、后坐、耳鸣、弹壳跳动、玻璃炸裂、掩体被打穿。

节拍：

```text
察觉威胁/先手暴露 -> 抢占掩体 -> 火力压制与换位 -> 一方弹匣空/卡壳/露头 -> 抓住空档突进或点杀 -> 结算伤亡与残局。
```

## 精神/意识战

适用：仙侠神识战、心魔、灵异、规则怪谈、克苏鲁、无限流、脑洞流、意志对抗。

关键要素：

- 先定规则：对抗发生在哪（识海、梦境、意识、规则层），输了会怎样（神魂受损、着魔、信息被夺、被同化）。
- 没有肢体，就用感知和规则的具体变化承载交锋：空间错乱、记忆被翻、声音失真、意志受压，且要写清它如何妨碍判断或行动。
- 抽象力量可以落到身体反应，但每次只选一两个有原因的结果，例如神魂受创导致动作停顿、规则反噬留下可见伤痕；不要堆通用疼痛或神态。
- 主角反制靠"稳住自我"：认清幻觉、守住一念、反向污染对手、找到规则漏洞。

节拍：

```text
精神接触/被拖入 -> 对手扭曲感知施压 -> 主角第一次动摇/受创 -> 抓住规则或守住本心 -> 反向压制 -> 一方神魂溃退，身体同步显伤。
```

## 大规模会战/军团

适用：历史争霸、军事、玄幻国战、宗门大战、兽潮、守城攻城。

关键要素：

- 不要平均写全场，用"主角视角的一小段战线 + 俯瞰几笔大局"交替，让读者既看到人也看到势。
- 大局用调度写：阵型、侧翼、援军、缺口、溃退、传令、旗号，一两笔交代胜负走向。
- 个人战和战局要挂钩：主角斩将/夺旗/破阵/守住某个点，直接改变大局某一处。
- 惨烈感靠规模落差：成片倒下、防线崩溃、以少拦多，别只盯着主角一个人爽。

节拍：

```text
两军对阵/压力逼近 -> 局部战线接战 -> 主角所在一点告急 -> 主角做出关键动作（斩将/破阵/守口） -> 牵动大局 -> 胜负倾斜与代价结算。
```
