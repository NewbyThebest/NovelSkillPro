# 对手压迫模板

让对手显强，不要只靠旁白说“他很强”。强要体现在主角的选择被压缩。

## 显强方式

- **封路：** 第一击不求命中，而是逼主角退到某个位置。
- **打要害：** 胸口、下颌、肋骨、膝侧、腕、喉、太阳穴。
- **连招无缝：** 第一拳落空，第二下已经堵住闪避路线。
- **读主角：** 主角刚用过的闪避方式，对手下一轮立刻针对。
- **逼规则：** 让主角继续躲会违规，不躲就受伤。
- **制造代价：** 即便没打中，也能刮裂衣服、震麻手臂、逼乱呼吸。
- **升级底牌：** 当普通攻击无效，亮出武核、禁术、兵器、法宝、阵法。
- **不降智失败：** 败在消耗、信息差、性格缺口、规则反噬或主角提前布局。

## 对手节拍

```text
先压身位 -> 再封退路 -> 逼主角交出应对 -> 记住应对 -> 下一轮针对 -> 主角付代价 -> 主角寻找反制。
```

## 性格带来的打法

- **狂傲型：** 喜欢正面压制和语言羞辱，容易被诱导过线或低估细节。
- **冷静型：** 少废话，试探后迅速调整，主角需要更精细的误导。
- **疯狗型：** 受伤不退，越打越凶，破绽在体力和节奏。
- **技术型：** 动作干净，封路精准，破绽在规则、环境或心理执念。
- **权势型：** 借裁判、规则、身份压人，爽点在被规则反噬。

## 压迫句式

- 他这一拳不是为了打中，是为了让祁临只能往左退。
- 第一拳刚空，膝盖已经顶住了祁临下一步的落点。
- 祁临躲开了拳，却没躲开拳风带来的麻意。
- 秦砾每落空一次，下一次就更贴近祁临刚才的退路。
- 真正麻烦的不是速度，是他每一下都知道自己要逼出什么。

## 注意事项

- 对手越强，主角越要付代价。
- 如果主角轻松躲开很多次，要解释原因：身法克制、对手保留、主角提前读招、场地限制。
- 对手的底牌要有代价，否则容易变成无限加强。
