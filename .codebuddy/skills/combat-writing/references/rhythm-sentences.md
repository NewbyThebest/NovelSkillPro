# 节奏与句式模板

打斗节奏不是一直快。快慢要跟信息类型匹配。

## 句式分工

- **短句：** 出手、命中、危险、判定、反转。
- **中句：** 动作因果、战术判断、位置变化。
- **长句：** 心理权衡、规则解释、战后余波。

## 高压近身战

适用：拳脚、短兵、训练室、巷战。

写法：

- 多用身体部位和距离：肩、肘、膝、腕、肋、半步、贴身、错肩。
- 动作要短促：冲、砸、扣、带、贴、滑、撞。
- 感官用闷响、破风、发麻、火辣、衣料撕裂。

句式：

```text
拳到面前。
祁临只偏了半肩。
没躲净。
衣袖炸开，手臂侧面立刻烧起一线痛。
```

## 大开大合战

适用：玄幻重击、兵器、怪物、战场。

写法：

- 先写势，再写点：风压、地裂、光芒、冲击，再落到伤口和动作。
- 每次大招后给环境后果。

句式：

```text
刀光压下来时，半座石台都暗了一瞬。
下一息，裂纹从他脚边炸开，沿着擂台一路爬向边缘。
```

## 斗法/异能战

适用：术法、武核、阵法、异能。

写法：

- 抽象能力必须落到物理结果。
- 不要只写光，写温度、声音、范围、代价、规则变化。

句式：

```text
红光不是亮起来的，是从秦砾眼底翻出来的。
他周围的空气像被拧紧，训练垫上的灰尘逆着风往上浮。
```

## 藏拙战

适用：主角要演弱。

写法：

- 高潮前写“能赢的清晰路径”。
- 随后用长一点的心理句说明为什么不能赢。
- 真动作要像失误，但读者能看出控制。

句式：

```text
这一拳若是落实，秦砾会倒。
记录也会亮。
祁临要的不是胜，是刚好不输。
```

## 移动端节奏

- 单段不要塞太多动作，2-4 行换一口气。
- 高潮处减少解释，先让动作发生。
- 解释规则和心理时，放在动作间隙或战后。
- 关键判定单独成段，如“时间到。”“越线，扣两分。”
