# 战术聪明感模板

用战术让打斗显得聪明。主角的强不只来自力量，也来自判断、误导、规则利用和代价计算。

## 战术来源

- **读动作：** 肩先动、脚先偏、呼吸先乱、兵器重心变化。
- **读规则：** 倒计时、越线、有效反击、不能杀、必须守、不能退。
- **读性格：** 对手暴躁、轻敌、好胜、怕输、爱炫技、急于证明。
- **读环境：** 墙、门、垫子、台阶、雨水、碎石、烟雾、灯光、死角。
- **读代价：** 对手爆发后的虚弱、禁术反噬、武核消耗、伤口牵动。

## 常用战术

- **诱敌过线：** 主角退到边缘，诱对手冲过头，借规则扣分或暴露空门。
- **借力带偏：** 不硬抗，用手腕、肘、肩、腰把对手冲势带歪。
- **以伤换节奏：** 接受擦伤，换取贴身、换位或打断呼吸。
- **打换气点：** 在对手爆发后的吸气瞬间击肋、撞胸、压喉。
- **攻支撑点：** 不打正面，打膝、踝、手腕、肩胛、兵器握点。
- **制造误判：** 前几次只躲，让对手以为主角不会反击；关键时刻突然进半步。
- **拆招化简：** 不完整使用大招，只拆出速度、卸力、步法或发力方式。
- **借环境反杀：** 利用墙面反弹、地面湿滑、碎石、窄门、光影遮挡。

## 写法公式

```text
观察一个细节 -> 判断一个规律 -> 承受一点压力 -> 利用规律做小反制 -> 让局势发生变化。
```

示例句式：

- 他没有看拳头，而是看秦砾的肩。
- 每次变向之前，对方的左脚都会先压半寸。
- 这一拳不能躲远，躲远就会被判消极。
- 主角要的不是打中，而是让对手冲势偏掉。
- 伤口火辣辣地疼，但节奏终于断了一拍。

## 避免降智

- 对手犯错要有原因：愤怒、时间压力、规则误判、体力下降、信息不足。
- 主角反制要有铺垫：前面看过、试过、吃过亏、故意引导过。
- 不要让主角突然知道一切。用“试错 -> 确认 -> 利用”写出过程。
