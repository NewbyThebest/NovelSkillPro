# 番茄末世题材

这是一个只服务 `末世` 的技能。

## 硬触发边界

只用于：

- 丧尸末世
- 灾变末世
- 冰河末世
- 囤货流
- 末世异能求生

## 末世最先看五件事

1. 书名有没有有效信息量
2. 世界设定是不是一眼能懂
3. 主角是不是适合末世
4. 主线是不是求生变强
5. 金手指有没有新意

## 稳妥起手

- 书名先点明末世
- 开局尽快抛出先机
- 先围绕生存写动作
- 物资、基地、队友、实力逐步扩大

## 人设和主线

新手更稳的人设通常是：

- 杀伐果断
- 先知先觉
- 不圣母
- 能快速做决策

前期主线常用：

- 囤物资
- 建避难所
- 收关键队友
- 抢关键机缘

## 创新重点

末世最适合创新的位置是：

- 金手指
- 灾变类型
- 资源体系
- 队伍结构

## 常见问题

- 书名没把卖点说出来
- 世界设定全是旧模板，没有一点变化
- 主角心软拖泥带水，和末世场景不匹配
- 只会杀丧尸，不会推主线

## 输出方式

1. 判断世界设定是否老套
2. 找主线是否清晰
3. 设计一处创新金手指
4. 补前五万字行动链
