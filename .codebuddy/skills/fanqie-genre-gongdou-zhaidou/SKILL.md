# 番茄宫斗宅斗

这是一个只服务 `宫斗 / 宅斗` 的技能。

## 硬触发边界

只在下面情况使用：

- 后宅争斗
- 宫廷妃嫔博弈
- 嫡庶、嫁娶、掌家、夺宠、站队
- 斗争本身就是主卖点

## 宫斗宅斗先看什么

先别急着堆坏人，先看：

1. 谁在争
2. 争什么
3. 输了会失去什么
4. 主角靠什么赢

如果这四件事不清楚，斗就会变成碎碎念和重复吵架。

## 斗点设计公式

`利益受损 -> 表面压制 -> 暗手布局 -> 反击翻盘 -> 地位变化`

宫斗宅斗的爽点不是单纯骂人，而是：

- 算计成功
- 局势反转
- 对方失势
- 主角上位

## 书名和开篇

宫斗宅斗类书名要尽量写出：

- 身份
- 斗争对象
- 反击钩子

开篇优先抛：

- 被算计
- 被夺资源
- 被压制
- 被逼到必须出手

## 常见问题

- 冲突太琐碎，不涉及核心利益
- 只会互呛，不会布局
- 主角反击太突兀，前面没有铺垫
- 配角全部一个坏法，没有层次
- 情绪很满，但斗争结果不改变局势

## 输出方式

1. 梳理势力和关系
2. 指出当前斗点值不值钱
3. 给一条更强的算计链
4. 如有需要，重做书名或开篇
