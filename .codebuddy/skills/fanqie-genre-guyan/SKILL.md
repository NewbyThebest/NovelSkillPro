# 番茄古言题材

把番茄课堂里偏 `古代言情` 的方法，整理成只服务古言的工作流。

## 硬触发边界

只用于：

- 古代言情
- 架空古言
- 侯门 / 王府 / 朝堂恋爱
- 古风婚恋、权谋恋爱

如果核心是宫斗宅斗算计，优先用 `fanqie-genre-gongdou-zhaidou`。

## 古言先看四个底盘

1. `身份差`
2. `礼法压制`
3. `关系束缚`
4. `情感推进空间`

古言不是换上古装词汇就行，必须让身份和规矩真的能影响剧情。

## 开书时优先定

- 女主处境
- 男主权力 / 身份
- 二人第一次强关系接触点
- 第一阶段情感和利益冲突

古言稳妥起手不是大段背景，而是：

`处境压迫 -> 关系碰撞 -> 利益牵扯 -> 情感留口`

## 古言常见好用抓手

- 婚约 / 替嫁 / 被迫同盟
- 家族利益和个人情感冲突
- 礼法压制下的小动作和反制
- 表面克制、内里拉扯

## 常见问题

- 只有古风词，没有古代关系压力
- 角色说话太现代
- 情感线推进全靠口头心动
- 配角全是功能人，家族结构不成立
- 开篇先讲世界，不先讲处境

## 输出方式

如果用户给设定：

1. 先定身份关系图
2. 再定核心冲突
3. 再定开篇碰撞点
4. 最后补情感推进节奏

如果用户给正文：

1. 看古言味道是否来自关系而非辞藻
2. 看男女主第一次博弈是否成立
3. 看家族 / 礼法是否真在起作用
4. 给修改建议
