# 番茄科幻题材

这是一个只服务 `科幻` 的技能。

## 硬触发边界

只在下面情况用：

- 黑科技文
- 星际文
- 软科幻升级文
- 科技文明升级

如果主轴已经变成末世求生，优先看 `fanqie-genre-apocalypse`。

## 科幻最重要的是新奇感

番茄课的核心不是要你严丝合缝做学术科幻，而是：

- 让读者看到新东西
- 让这些新东西形成升级爽点

## 科技树怎么用

科技树要承担的，本质上就是玄幻里的升级体系。

至少要有：

- 层级
- 前置科技
- 解锁回报
- 力量转化
- 新地图

## 两条常用主线

1. `黑科技都市`：科技升级 -> 社会变化 -> 力量碾压
2. `星际扩图`：文明升级 -> 开新星球 -> 新种族 / 新副本

## 常见问题

- 只会堆术语，不会转成爽点
- 科技升级后读者感受不到变强
- 星际只有背景，没有地图差异
- 过度追求现实合理，反而失去网文化节奏

## 输出方式

1. 判断是黑科技还是星际
2. 搭科技树骨架
3. 指出每一级如何兑现爽点
4. 补新奇感和地图变化
