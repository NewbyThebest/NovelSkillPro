# 番茄世情爽文

这是一个只服务 `世情爽文` 的技能。

## 硬触发边界

只在下面情况用：

- 家庭关系矛盾
- 婚姻冲突
- 亲情压迫和反击
- 现实题材的强情绪打脸短长篇

## 世情爽文先看导语

导语必须：

- 具象
- 和主角核心利益有关
- 快速积累情绪

不要泛泛描述“她一直受委屈”，要给一个能让人一下代入的具体场景。

## 冲突升级

`小委屈 -> 核心利益受损 -> 情绪爆点 -> 反击 -> 更大后果`

如果一开始只是鸡毛蒜皮，后面却突然离婚断亲，读者会觉得假。

## 常见问题

- 导语重复书名信息
- 冲突和主角核心利益不相连
- 极端转折没有足够积累
- 打脸只停留在吵架层

## 输出方式

1. 找最具体的切入口
2. 校准利益冲突
3. 设计情绪累积链
4. 补一版更强导语或开篇
