# 番茄仙侠题材

把番茄课堂里关于 `仙侠入门 / 凡人流创作 / 仙侠常见误区` 的内容，压成一个只服务仙侠题材的工作流。

## 硬触发边界

只有在下面情况才用：

- 凡人流
- 古典仙侠
- 日常风仙侠
- 洪荒西游向仙侠
- 明确以修仙、宗门、资源争夺、求长生为主轴的作品

下面情况不要误用：

- 传统玄幻，转 `fanqie-genre-xuanhuan`
- 都市修真但核心是都市爽文，优先看 `fanqie-genre-urban`
- 脑洞签到修仙，若核心卖点是系统套路而非仙侠底盘，可配合 `fanqie-genre-brainhole`

## 先判断是哪一路仙侠

先帮用户定流派，再给建议：

1. `凡人流`：主角资质普通，靠机缘、智商、苟发育慢慢崛起
2. `古典仙侠`：门派、秘境、夺宝、儿女情仇、仙人江湖
3. `日常风仙侠`：师门日常、轻松幽默、关系感强
4. `洪荒 / 西游 / 大背景神话流`：先天自带大舞台和设定包袱

如果流派没定，先不要急着改文，先把流派定清。

## 仙侠题材的核心抓手

- 先有 `修炼目标`，再有剧情
- 先有 `资源争夺`，再有冲突
- 先有 `成长路径`，再有爽点
- 先让读者看见 `主角怎么活下去 / 变强`，再慢慢铺大世界

仙侠的底盘不是空讲仙气，而是：

`长生欲望 + 修炼体系 + 资源矛盾 + 个人成长`

## 开篇怎么做

优先遵循下面几条：

1. 凡人流少写凡间平淡日常，尽快切到 `奇遇 + 入修行路`
2. 开头要简单，不要先堆一排人物、宗门、设定词
3. 情节优先，环境意境次之
4. 先给读者一个明确期待：
   - 主角要进宗门
   - 主角要抢机缘
   - 主角要脱离必死局
   - 主角拿到金手指但要立刻变现

## 人设与金手指

番茄仙侠更稳的人设是：

- 聪明
- 果断
- 知道趋利避害
- 不轻易犯蠢

凡人流常见安全做法：

- 灵根差
- 出身低
- 金手指偏辅助
- 升级不飞天，但能持续给回报

金手指优先做成：

- 加速修炼
- 提纯资源
- 推演功法
- 复制或培育灵药

少做一上来就把仙侠写成无脑打脸。

## 常见误区

- 把仙侠写成玄幻打脸文
- 开篇只顾氛围，不顾抓力
- 主角不聪明，却硬写残酷修仙世界
- 设定名词太多，读者进不去
- 误以为仙侠一定要靠大脑洞，忽视情节和成长
- 群像太早上量，新人自己先控不住

## 输出方式

如果用户给的是设定：

1. 判断流派
2. 定主角底盘
3. 定第一阶段修行目标
4. 定开篇切口

如果用户给的是正文：

1. 判断更像哪一路仙侠
2. 找开篇是否拖
3. 找升级线 / 资源线 / 冲突线是否成立
4. 给改法或样段

## 与其他技能协同

- 开篇第一页留人不够，接 `fanqie-opening-hook`
- 单章节奏发虚，接 `fanqie-pacing-conflict`
- 长篇中段修仙升级写疲了，接 `fanqie-longform-control`
