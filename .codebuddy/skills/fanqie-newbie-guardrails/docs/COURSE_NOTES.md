# 课程拆解笔记

来源：

- 番茄课堂《新手写作五大避雷针｜让你的故事瞬间“上头”》
- 番茄课堂新手高频问题总课（开篇、世界观、人设、视角、剧情、文笔）
- 番茄课堂《网文选题秘诀：精准避坑，打造你的专属风格！》
- 番茄课堂《题材怎么选？开篇怎么写？人设怎么立？三大常见问题，本节为你拆解》
- 番茄课堂《拉新向创作方法与技巧，新颖设定、精铸开篇、看点打造等》

这份笔记只保留可执行的方法，不复述课堂废话。

## 一、选题与题材

### 题材与内容契合

- 书名、标签、简介、正文要对齐
- 题材核心爽点要兑现，不然读者会觉得被骗进来
- 故事感要强，别把题材写成科普或流水日常
- 对新手来说，先写自己喜欢且擅长的大题材，比盲追热点更稳

### 设定微创新

- 新颖设定是拉新文的重要抓手
- 如果市场上已有大量同类梗，不能只做表面换皮
- 旧瓶装新酒的关键是：保留有效期待线，同时给读者新的打开方式
- 例如信息差、掉马这类老框架，仍然能打，但必须换身份关系、悬念组织或反应链

### 题材辨识度

- 年代、历史、玄幻、仙侠、系统、职业等题材，都要有自己的专属识别点
- 识别点可以是时代细节、体系规则、场景道具、职业高光

### 跨题材混搭

- 混搭不是拼盘，核心是逻辑纽带
- 新手更稳的做法是先把单一题材写扎实

### 题材选择补充

- 先看自己平时长期看什么题材
- 再看脑海里最容易冒出来的故事属于什么题材
- 这里说的题材优先是 `现言 / 古言 / 玄幻 / 架空 / 都市脑洞 / 总裁 / 系统 / 悬疑` 这类大方向，不是更细的小梗

## 二、写作人称

- 第二人称基本不讨论
- 新手默认第三人称更稳，更适合练长篇
- 第一人称适合更强代入、偏新媒体感、总字数不太长的故事
- 一旦定人称，必须全程统一

## 三、避免流水账

流水账的本质不是“事情少”，而是“没有期待感”。

### 核心补法

1. 给普通行为补前因后果，形成冲突
2. 给角色补着急、憋屈、害怕、愤怒等强情绪

### 结论

- 冲突让情节有拉力
- 情绪让读者愿意代入

## 四、开篇不吸引人

### 开篇太寡淡

- 常见问题：从最无聊的地方写起
- 更优思路：从最有噱头、最出事、最能吊胃口的位置切入
- 适合新手的冲突，往往不是“大而空”的生死危机，而是小范围、强关系、强代入的人性抉择

### 开篇太复杂

- 新手开篇不宜场景过多、人物过多、支线过多
- 更优做法：先用一个核心事件把主角和主要矛盾打亮

## 五、人设有问题

### 人设扁平化

- 所有人说话方式一样
- 所有人发火方式一样
- 人物只有标签，没有独特行为模式

### 人设前后不统一

- 根源往往是大纲支撑不住，或作者生活素材和判断力不够
- 人设变化要靠剧情推动，不能无缘无故转性

### 人设补充

- 人物小传只是梗概，不是人设自动成立的证明
- 真正的人设要靠剧情碰撞、配角拉扯、极端选择中的反应去立

## 六、世界观与视角

### 世界观

- 设定不能又糊又灌
- 更好的做法是场景化、行动化、对话化地露出规则
- 先交代最关键的规则，别把百科全书放在开头

### 视角

- 同一章节不要频繁乱切
- 新手默认顺叙更稳
- 多人场景先保一个主视角，别人人都抢镜

## 七、主线与剧情

- 开篇要尽快明确主角目标和主线方向
- 主线需要持续有阻碍，不然故事就会散和平
- 爽点不是突然亮一下，而是靠铺垫、压迫、反击组成
- 情节转换要自然，不能上一秒森林、下一秒闹市却毫无过渡
- 打斗、追杀、逃生只是场景，不是目标；目标不亮出来，读者就不知道为什么要继续看

## 八、核心梗发散

- 当用户只有一个强梗时，可以把梗里的关键人物分别当作圆心去扩剧情
- 从每个人的利益、欲望、恐惧、秘密、误判出发，让同一事件长出多股张力
- 这是从“有一个点子”走向“能撑起故事”的有效办法

## 九、其他问题

### 秀文笔

- 网文先讲清楚故事，再考虑漂亮
- 形容词贵精不贵多
- 排版也属于可读性的一部分，手机端不要堆厚墙

### 钩子

- 钩子就是悬念、伏笔、包袱、危机
- 章节结尾最好留小钩子

### 书名和简介

- 书名是题材和卖点的第一眼定位
- 简介像产品宣传语，要提炼主线、冲突或名场面

## 十、课程的底层倾向

这节课的核心偏好很明确：

- 重点击
- 重开篇
- 重追读
- 重读者感受
- 重简单直接

它不是文学性训练，而是平台化写作训练。
