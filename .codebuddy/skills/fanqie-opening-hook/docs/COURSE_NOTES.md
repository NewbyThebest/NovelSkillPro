# 课程拆解笔记

来源：番茄课堂《开篇切入点的选择、剧情切入的关键要点》

补充吸收：番茄课堂《拉新向创作方法与技巧，新颖设定、精铸开篇、看点打造等》

## 一、两种主流开篇

### 剧情切入

- 直接抛当前危机、阻力、问题
- 优点是简单、直接、对新手友好

### 场景切入

- 先用环境和氛围把读者拉进去
- 吃文笔、吃控制力，不适合多数新手直接用

### 课程结论

- 新手优先剧情切入

## 二、先有主线，再找切口

- 整本主线最好能一句话概括
- 再整理第一卷卷纲
- 开篇不一定从最早发生的事开始，而是从最适合引爆的事开始

## 三、主动引爆

- 作者不能等剧情自己热起来
- 要主动把炸点提到开头
- 让读者即使不懂前情，也会被当前事件吸住

## 四、好奇升级

- 不是单纯抛一个钩子
- 而是通过小细节不断升级读者的好奇

推荐链条：

`小异常 -> 更怪的细节 -> 真正引爆`

## 五、精铸开篇补充

- 开篇要精，字字尽量有功能
- 从第一个字开始就最好给出抓手
- 三段话以内最好有重点
- 300 字内尽量进入主题
- 任何拖节奏、让读者枯燥的句子，都应该删或改

## 六、反差和锚点

- 现实与幻想的反差、正常与异常的反差，都很抓人
- 角色要有一句话能记住的点
- 锚点能帮主角快速立住

## 七、角色与信息控制

- 前五章不要出现没有意义的人
- 开篇越简单越好，但必须有主线、有能动性
- 读者需要快速知道：主角什么处境，要干什么，为什么非干不可

## 八、前三章节奏参考

- 第一章：先抛钩子
- 第二章：推进 + 加伏笔
- 第三章及后续：第一次高潮 / 爽点 / 新坑

## 九、底层倾向

- 这是一门强实战、强留存的开篇设计课
- 核心不是文艺，而是切口、引爆、升级、追读
