# 课程拆解笔记

来源：番茄课堂《节奏混乱？冲突乏力？爽点不足？你的问题，本篇为你解决！》

补充吸收：番茄课堂《拉新向创作方法与技巧，新颖设定、精铸开篇、看点打造等》

## 一、节奏混乱

- 节奏问题常见于：不进主线、转场突兀、规则打架
- 最直接的处理办法不是“写顺一点”，而是删掉无效内容
- 合理布局的本质，是在关键节点放冲突和悬念

## 二、冲突乏力

- 没阻碍，故事就不成立
- 高潮前没有伏笔，高潮就会空
- 高潮后没有余波，高潮就会像孤立事件

### 课程里很关键的一点

- 高潮前 3 到 5 章开始埋线
- 高潮后 1 到 2 章补后果和变化

## 三、爽点不足

- 爽点不是只有打脸
- 爽点可以是升级、逆袭、智斗、情感回报
- 爽点不能只有“结果”，还要有压制、反击、反馈

### 推荐结构

`小冲突 -> 快解决 -> 强反馈`

以及：

`抑 -> 扬 -> 反馈`

例如：

- 抑：被轻视、被卡、被羞辱、被误会
- 扬：快速反击、反超、揭穿、翻盘
- 反馈：他人震惊、身份提升、资源到手、关系改写

### 拉新密度补充

- 300 字尽量给 1 个小看点
- 500 字尽量给 1 个小钩子
- 1 章尽量形成 1 个完整剧情单元，并在结尾留悬念

## 四、这节课的底层倾向

- 它是强平台导向的改稿课
- 核心不是文学打磨，而是留读者、保追更、提爽感
- 它非常适合拿来改单章、调小高潮、补高潮后果
