# 番茄末世题材

## 一、边界与易混淆

只服务末世。适用：

- 丧尸末世
- 灾变末世
- 冰河末世
- 囤货流
- 末世异能求生

## 二、流派/分类选型

末世下笔前先看五件事，这五条决定这本立不立得住：

1. 书名有没有有效信息量
2. 世界设定是不是一眼能懂
3. 主角是不是适合末世
4. 主线能否从求生决策延伸到资源、基地、队伍或生存秩序的变化
5. 金手指有没有新意

## 三、核心爽点/抓手

末世的爽 = **绝境求生 + 先机变现 + 局面扩张**。读者看的是主角在崩坏世界里把先机转成可见的资源、关系或安全空间。

稳妥起手：

- 书名先点明末世
- 开局尽快抛出先机
- 先围绕生存写动作
- 物资、基地、队友、实力逐步扩大

## 四、开篇切口

- 书名先把末世卖点说出来。
- 开局尽快抛先机（先知、异能、囤货优势），别拖。
- 先写生存动作，让读者立刻看到"怎么活下去"。

## 五、人设与金手指

新手更稳的人设：

- 能辨认风险和资源优先级
- 先知或异能要转成具体行动
- 保护谁、放弃什么要有明确边界
- 高压下能做决策，并承担决定的后果

前期主线常用：

- 囤物资
- 建避难所
- 收关键队友
- 抢关键机缘

**末世最适合创新的位置**（同质化重灾区，优先在这里做新）：

- 金手指
- 灾变类型
- 资源体系
- 队伍结构

## 六、常见误区

- 书名没把卖点说出来
- 世界设定全是旧模板，没有一点变化
- 主角心软拖泥带水，和末世场景不匹配
- 只会杀丧尸，不会推主线

## 设定阶段必备维度

按顺序确认：

1. 灾变来源、发生范围和当前阶段，哪些事实已经公开，哪些仍在变化。
2. 主角的生存优势、即时目标和不能失去的对象。
3. 物资、交通、安全区或基地的稀缺规则，以及资源如何转成地位和选择。
4. 队伍、基地或幸存者组织的合作条件、分配方式与冲突边界。
5. 丧尸、灾害、异能或其他外部威胁的可观察规则和升级方式。
6. 金手指、先知或特殊资源的使用门槛，不能随意覆盖生存成本。
