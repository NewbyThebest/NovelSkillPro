# 黄金三章审核标准文档

> 依据"悟道课·黄金三章核心要点"整理，适用于网络小说开篇章节（通常前3-5章）的专业审核。

---

## 审核维度说明

共 **10 个维度**，每项 **10 分制**，总分 100 分。

---

## 维度一：注意力管理 / 读者钩子（10分）

**核心原理**：写作是管理读者注意力的科学。手机阅读时代，读者对新书的耐心仅有三次屏幕滑动（约200-300字）。

| 分值 | 标准描述 |
|------|----------|
| 9-10 | 开篇即有强烈的悬念、冲突或情绪冲击，前200字内读者不可能停止阅读 |
| 7-8  | 开篇有较明确的钩子，但需要500字以内才能真正抓住读者 |
| 5-6  | 有一定吸引力，但钩子模糊，读者需要一定耐心才能继续 |
| 3-4  | 开篇平淡，缺乏明确的悬念或情绪驱动 |
| 1-2  | 开篇完全无法抓住读者，信息堆积，与读者情绪完全隔离 |

**扣分项**：
- 大场景开头（多角色、多事件同时登场）
- 与主角无关的前置剧情
- 抽象数字描述（"十万大军"、"百万敌军"等）

---

## 维度二：信息颗粒度（10分）

**核心原理**：低颗粒度信息（年代、地名、等级体系说明）消耗读者耐心；高颗粒度信息（声音、动作、具体细节）刺激感官。

| 分值 | 标准描述 |
|------|----------|
| 9-10 | 所有信息均通过具体的感官细节传达，无需读者主动动脑理解 |
| 7-8  | 信息大多具体，偶有少量需要理解的抽象概念但影响不大 |
| 5-6  | 具体细节与抽象信息混杂，读者需消耗一定注意力 |
| 3-4  | 大量抽象信息、概念解释，信息堆积明显 |
| 1-2  | 开篇几乎全是设定说明、背景铺垫，完全是信息堆砌 |

**扣分项**：
- 自创年代名/地名密集出现且无上下文支撑
- 等级体系、功法门派集中介绍
- "这是一个……的世界"式直接背景说明

---

## 维度三：折射技法（聚焦度）（10分）

**核心原理**：折射——把宏大的场景、史诗感的东西聚焦到具体的一个点（一件道具、一个动作、一段对话）上，通过小切面烘托大世界。

| 分值 | 标准描述 |
|------|----------|
| 9-10 | 用极精准的一个或两个具体切点折射出宏大背景，读者无需额外说明即可感知世界规模 |
| 7-8  | 有折射手法，但切点选取不够精准，仍有部分直接叙述 |
| 5-6  | 折射与直接描写并存，聚焦感不足 |
| 3-4  | 基本以全景式大场景叙事为主，缺乏聚焦 |
| 1-2  | 完全俯瞰式上帝视角，读者成为"看热闹的上帝"，无法代入 |

**参考标杆**：
- 优秀：庆余年·瞎子+竹篓+黑骑对峙；诡秘之主·左轮+笔记本+镜子
- 合格：聚焦到具体人物的具体危机时刻
- 失败：直接描写"敌军十万""禁咒大战"

---

## 维度四：情绪冲击力（10分）

**核心原理**：恐惧是最基础、最容易操控的情绪，最适合开篇建立情绪冲击。好的开篇所有描写应指向同一个情绪目标。

| 分值 | 标准描述 |
|------|----------|
| 9-10 | 开篇情绪明确、强烈，读者能清晰感受到恐惧/紧张/好奇/期待中的至少一种，且持续贯穿 |
| 7-8  | 情绪有明确方向，但强度不够或有中途断裂 |
| 5-6  | 情绪方向模糊，读者需主动揣摩作者意图 |
| 3-4  | 情绪散乱，多种情绪混杂无主次 |
| 1-2  | 开篇无情绪，纯叙事/说明文式写法 |

**扣分项**：
- 情绪描写与实际描写脱节（告诉读者"很恐惧"而不展示恐惧的细节）
- 大场景中情绪失焦

---

## 维度五：临场感与感同身受（10分）

**核心原理**："认知、临场感、感同身受"——读者要能看见、摸得着、想象得到；不要让读者成为上帝，要把读者的视野拉到地上，让他看到主角每根手指的颤动。

| 分值 | 标准描述 |
|------|----------|
| 9-10 | 读者完全沉浸在主角视角，感官细节丰富，如同身临其境 |
| 7-8  | 临场感较好，但部分细节不够具体或视角偶有跳出 |
| 5-6  | 有一定临场感，但上帝视角与主角视角频繁切换 |
| 3-4  | 主要为客观叙述，读者与主角情感距离较大 |
| 1-2  | 完全第三人称全知视角叙事，读者是旁观者 |

**加分项**：
- 通过主角的具体动作、生理反应传达情绪（咬嘴唇、屏住呼吸、手指颤抖）
- 用可感知的声音、气味、触觉细节增强沉浸感

---

## 维度六：设定物化（道具化）（10分）

**核心原理**："写故事而不是写设定"——把设定剧情化或物化。一把左轮手枪胜过一千字的工业背景说明；月亮是红色的胜过"这个世界有诡异的超凡力量"。

| 分值 | 标准描述 |
|------|----------|
| 9-10 | 所有世界观要素均通过故事内的道具、事件、对话自然呈现，零设定说明文字 |
| 7-8  | 大部分设定通过故事呈现，有少量必要的背景说明但处理得当 |
| 5-6  | 设定与故事各占约一半，读者需记住部分设定才能理解剧情 |
| 3-4  | 设定占主导，故事为设定服务 |
| 1-2  | 开篇即是设定说明书，大量篇幅用于解释体系/世界观/历史 |

**最高标准参考**：读者看完前三章，无需记住任何专有名词，仍能被故事完全吸引。

---

## 维度七：减法原则（叙事克制度）（10分）

**核心原理**：大多数人习惯做加法（1+1+1+1），写作要做减法——让读者注意力越来越聚焦，而非越来越分散。

| 分值 | 标准描述 |
|------|----------|
| 9-10 | 每一句话都在服务同一个叙事目标，无冗余信息，节奏感强 |
| 7-8  | 叙事较克制，但有少量可删减的冗余描写 |
| 5-6  | 叙事中度冗余，部分场景或人物介绍可大幅精简 |
| 3-4  | 冗余信息较多，读者注意力被分散到多个不重要的点上 |
| 1-2  | 大量冗余，每一段都在做加法，整体注意力极度分散 |

**典型问题**：
- 同时介绍多个次要角色（开篇除主角外出现超过3个有名字的角色）
- 场景切换过于频繁
- 插叙/倒叙破坏了主线情绪节奏

---

## 维度八：流派承诺兑现（10分）

**核心原理**：每个流派本质上是向读者表露一种"危机"并做出相应"承诺"——书名、简介与开篇内容构成与读者的契约关系。

| 流派 | 危机 | 承诺 |
|------|------|------|
| 苟道流 | 世界危险不安定 | 主角能安全生存 |
| 探索流 | 世界充满未知 | 有趣的探索过程 |
| 凡人流 | 主角弱小无助 | 必然逆袭变强 |
| 爽文流 | 主角受到压制/委屈 | 大规模反击爽点 |

| 分值 | 标准描述 |
|------|----------|
| 9-10 | 开篇的危机类型与书名/简介/类型的承诺高度吻合，读者期待与内容完全匹配 |
| 7-8  | 基本吻合，但承诺的兑现路径不够清晰 |
| 5-6  | 部分吻合，危机或承诺有一方不够明确 |
| 3-4  | 危机或承诺其中一方缺失 |
| 1-2  | 开篇内容与书名/简介/类型严重不符，读者会感到被"耍" |

---

## 维度九：主角吸引力与关注聚焦（10分）

**核心原理**：开篇最重要的目标之一是把读者的注意力"钉"在主角身上。读者必须对主角产生好奇、担忧或认同感。

| 分值 | 标准描述 |
|------|----------|
| 9-10 | 主角第一次出场即有鲜明的特质或处境，读者立即对其产生强烈的好奇或代入感 |
| 7-8  | 主角形象较清晰，读者有基本的关注，但吸引力不够强烈 |
| 5-6  | 主角存在感一般，与配角或场景平分秋色 |
| 3-4  | 主角戏份少或形象模糊，读者注意力没有集中到主角身上 |
| 1-2  | 主角登场延迟，或在大量其他角色/场景中完全淹没 |

**加分项**：
- 利用"反逻辑"设计突出主角（强大一方向弱小一方退让）
- 主角处于高压危机中，读者自然关注其命运

---

## 维度十：悬念密度与可读性（10分）

**核心原理**：当读者的视野被限制到一个具体的地方，他们会对看不见的东西产生好奇、担忧、期待——这种"视野外的悬念"是维持阅读动力的核心。

| 分值 | 标准描述 |
|------|----------|
| 9-10 | 每章结尾或关键节点都有明确的悬念钩子，读者看完一章必须看下一章 |
| 7-8  | 有一定悬念设计，但部分章节结尾悬念感不足 |
| 5-6  | 悬念零散，主要靠情节推进而非悬念驱动阅读 |
| 3-4  | 几乎无悬念设计，全靠读者主动兴趣维持 |
| 1-2  | 完全无悬念，信息透明，读者没有继续阅读的驱动力 |

**悬念类型参考**：
- 身份悬念（主角/神秘人物的真实身份）
- 命运悬念（主角在危机中能否脱险）
- 世界悬念（某件道具/现象/规则的真相）
- 关系悬念（角色间的真实关系与目的）

---

## 综合评级标准

| 总分 | 评级 | 含义 |
|------|------|------|
| 90-100 | S级·编辑直通 | 极具潜力，可直接进入签约审核流程 |
| 80-89  | A级·推荐修改后上线 | 整体质量较高，针对性修改后可发布 |
| 70-79  | B级·需要较大调整 | 有亮点但问题明显，建议重写关键章节 |
| 60-69  | C级·建议重写开篇 | 问题集中，整体结构需要重新设计 |
| 60以下 | D级·需系统性重写 | 未达到基本的网文开篇标准，建议推倒重来 |

---

## 修改意见框架

针对每个评分维度，修改建议应包含：
1. **问题定位**：具体指出问题所在的段落/句子
2. **原理说明**：解释为什么这是问题（对应哪条黄金三章原理）
3. **修改方向**：给出具体可操作的修改思路
4. **示范参考**：如适用，提供类比的优秀案例（如庆余年、诡秘之主的处理方式）
