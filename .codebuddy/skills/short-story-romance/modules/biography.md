# 模块：人物小传（Character Biography）

> 在第三步·人物小传（硬性关卡）时加载。为每一个主要角色生成≥2000字的前史生平。
> 
> **同步加载**：`modules/psychology.md`（心理学·社会学·原生家庭理论参考）

## 核心原则

人物小传不是角色卡片的扩写，而是**性格成因的考古**。每一个决断习惯、每一种情绪反应模式，都必须在小传中有根可循。后续正文中角色面对冲突时的选择，必须能反向追溯到小传里的具体经历。

**心理学是透视镜**：用依恋理论、防御机制、原生家庭理论来理解角色"为什么这样做"，但在文本中**绝不出现术语**——读者看到的是具体行为和画面，不是心理学名词。

**一句话检验标准**：读完小传后，读者应该能预测这个人物在两难抉择中"更可能"做什么——但又不完全确定。

|------|------|------|

## 小传结构模板（每人≥2000字）

```
# [角色名] 人物小传

## 一、基础画像（200-300字）
- 姓名/年龄/外貌特征（限3句，挑最能暗示性格的）
- 当前身份与表面形象
- 真实内核（与表面的落差）
- 核心欲望 vs 核心恐惧

## 二、家庭环境（400-500字）
[理论透镜：依恋理论→四种依恋类型的童年根源 | 原生家庭理论→五种有毒家庭模式 | 代际传递 → psychology.md §一、§三]
- 原生家庭结构（父母关系、兄弟姐妹、经济状况）
- 家庭氛围（温暖/冷漠/高压/混乱/缺失）→ 对应哪种家庭模式？
- 父母的教养方式对性格的塑造 → 形成了哪种依恋类型？
- 家庭中的权力关系（谁说了算？谁被忽视？）→ 角色在家庭中的固化角色是什么（照顾者/替罪羊/金童/隐形人）？
- 童年中最深的一个画面（具体场景，不是概述）→ 这个画面应该能"一眼看出"依恋类型的成因

## 三、所受教育（300-400字）
- 教育经历（正规教育/自学/师承/特殊训练）
- 教育过程中的关键人物（恩师/对手/同窗）
- 教育赋予的能力与认知框架
- 教育经历中埋下的价值观或执念

## 四、重大事件（每个事件300-400字，≤3个）
[理论透镜：创伤心理学→4F反应（Fight/Flight/Freeze/Fawn）| 习得性无助 | 创伤联结 → psychology.md §四]

### 事件一：[命名——用一个画面或动作命名，不用抽象词]
- **发生了什么**（具体场景、具体动作、具体对话）
- **她/他做了什么选择**（以及为什么——回扣家庭/教育的影响）→ 这个选择对应哪种创伤反应模式（Fight/Flight/Freeze/Fawn）？
- **不可逆后果**（这件事永久改变了什么）
- **性格烙印**（从此以后，她/他在面对___时，会本能地___）→ 这里形成了哪种防御机制？

### 事件二：[命名]
（同上格式）

### 事件三：[命名]（可选）
（同上格式）

## 五、性格决断图谱（200-300字）
[理论透镜：防御机制表 | 依恋类型的关系表现 | 认知失调 → psychology.md §二、§一、§五]
基于以上经历，提炼该角色的决断模式：
- 面对背叛 → 她/他会___（因为事件X教会了她/他___）→ 启动哪种防御机制？
- 面对示弱 → 她/他会___（因为家庭中___）→ 依恋类型如何影响反应？
- 面对权力 → 她/他会___（因为教育中___）
- 面对爱情 → 她/他会___（因为事件Y让她/他___）→ 依恋类型如何影响亲密关系模式？
- 底线是什么 → 触碰底线后的反应模式 → 所有防御机制失效后的"裸反应"是什么？
- **灰度地带** → 在什么情况下以上模式会失效？（这是正文中制造反转的弹药）

## 六、语言风格DNA（100-150字）
- 口头禅/高频句式
- 愤怒时的语言特征（短句？沉默？反问？）
- 脆弱时的语言特征（转移话题？自嘲？攻击？）
- 说谎时的语言特征（细节过多？眼神回避？转守为攻？）

## 七、心理学内部标注（不公开，供创作参考）
- 依恋类型：[安全/焦虑/回避/混乱]（源自：___）
- 主要防御机制：[从防御机制表中选取2-3个]
- 原生家庭模式：[从五种有毒家庭模式中选取，或标注"健康"]
- 创伤反应默认模式：[Fight/Flight/Freeze/Fawn]
- 觉醒/崩塌/暴露的心理学弧线：[简述角色的心理转变路径]
```

## 写作铁律

### 铁律1：具体事件，不要概述
- ❌ "她从小缺少关爱" → ✅ "她六岁生日那天自己做了一碗面，等到面凉了也没人回来"
- ❌ "他性格强势" → ✅ "他七岁时看着父亲跪在债主面前，发誓自己绝不会跪任何人"

### 铁律2：因果链必须闭合
每个性格特征都必须能追溯到具体事件。正文中角色的每个关键决断都必须能在小传中找到根源。
```
家庭环境/教育/重大事件 → 性格烙印 → 决断模式 → 正文中的具体行为
```

### 铁律3：重大事件≤3个，但每个都必须是"分水岭"
- 事件必须是**不可逆**的——经历之后，这个人再也回不到之前的状态
- 事件之间必须有**递进或转向**关系——不是三个平行的创伤，而是一条性格锻造链

### 铁律4：性格决断图谱必须有"灰度"
- 不要写成"她永远勇敢/他永远懦弱"
- 要写出**在什么情况下她的勇敢会失效**、**什么样的示弱能突破他的防线**
- 这些灰度地带就是正文中制造冲突和反转的弹药

### 铁律5：2000字是底线，不是目标
- 核心角色（女主/男主）建议2000-2500字
- 信息密度优先：每一段都在为正文储备"弹药"，不允许注水

## 小传与正文的衔接机制

小传完成后，第四步章纲为每章直接填写"性格驱动"字段（基于小传素材一次性写完，不存在回填流程）；第五步正文创作中执行以下规则：

### 章纲衔接
章纲每章必填项包括：
```
- 性格驱动：[本章核心冲突触发了哪个角色的哪条性格烙印？该角色基于小传会做出什么决断？]
```

### 正文衔接
1. **每个关键决断必须有小传根源**：角色在重要抉择时的行为，必须与小传中的性格决断图谱一致（或有合理的突破/成长理由）
2. **对话风格锚定小传**：角色在不同情绪状态下的语言表现，必须符合小传§六的语言风格DNA
3. **冲突设计基于小传灰度**：最强的冲突来自角色的"灰度地带"——正是小传中标注的"在___情况下会失效"的那些点

### 审核新增项
自动审核清单增加：
- [ ] 本章角色关键决断是否有小传依据？
- [ ] 角色语言风格是否符合小传§六？
- [ ] 冲突是否利用了角色的灰度地带？
