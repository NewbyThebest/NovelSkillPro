## 女主原型

**核心特征**：隐形能力者，有底线触碰后不可逆。
**心理学底色**：焦虑型依恋（觉醒前）→ 安全型重建（觉醒后）| 原生家庭常见模式：情感忽视型/亲职化 | 默认创伤反应：Fawn(讨好) → 觉醒后切换为 Fight/Flight

| 能力类型 | 示例 | 适配题材 |
|---------|------|---------|
| 法律/商业 | 律师/财务/公司联合创始人 | 都市情感 |
| 医术 | 外科医生/草药师/护理专家 | 医疗/重生 |
| 设计/艺术 | 建筑师/设计师/画家 | 都市/知识产权 |
| 战略/情报 | 战略顾问/黑客/调查员 | 悬疑/身份反转 |
| 战斗力 | 格斗/Alpha形态/特殊血脉 | 狼人/ABO |
| 血脉/身份 | 隐藏贵族/真正继承人/皇室 | 狼人权力/古代 |

**变体**：受害者本人 / 双胞胎姐妹 / 重生者 / 隐藏贵族

**攻击性等级**：

| 等级 | 描述 | 表现 |
|------|------|------|
| L0 | 无法攻击 | 已死/失忆/临终 |
| L1 | 冰冷克制 | 沉默离开/自我牺牲 |
| L2 | 冷幽默型 | 内心犀利，行动干脆 |
| L3 | 隐忍后爆发 | 前期压抑，一次性倾泻 |
| L4 | 刀刀见血 | 全程语言锋利/公开碾压 |
| L5 | 暴力复仇 | 直接使用肢体暴力/武器 |

## 男主（被离开的）原型

**核心缺陷**：把女主当理所当然 + 拒绝相信女主的话
**心理学底色**：回避型依恋 | 原生家庭常见模式：控制型/溺爱型 | 主要防御机制：否认+合理化+投射 | 低自我分化（容易被三角化操控）

**被操控的盲点类型**：

| 类型 | 机制 | 对立方利用方式 |
|------|------|-------------|
| "她好可怜"型 | 对立方伪装弱者，激发保护欲 | 每次男主想去找女主，对立方就"生病/哭/出事" |
| "反叛控制"型 | 男主把对女主的感情误读为被迫 | 对立方暗示"你是被她绑住的" |
| "旧情难忘"型 | 男主对初恋/旧情有执念 | 对立方就是初恋或利用初恋记忆 |
| "兄妹情谊"型 | 男主把对立方当妹妹/家人 | 对立方利用这层身份做不被怀疑的事 |

**自欺六阶段**（= 防御机制逐层击穿 → `psychology.md` §二）：
否认("她在闹脾气") → 合理化/自信("她离不开我") → 焦虑(电话不通，回避策略首次失效) → 投射/愤怒("她怎么敢"=把自己的恐惧投射为愤怒) → 恐慌(发现物证链，认知失调无法维持) → 崩溃/退行(膝盖撞地板，所有防御全线崩溃)

## 对立方原型（九种）

**心理学底色**：混乱型依恋 | 原生家庭常见模式：金童型/暴力型/情感忽视型 | 主要防御机制：理想化+贬低、反向形成 | 核心工具：三角化（拉男主进入三角关系）+ 光环效应（利用外表/弱态维持伪装）

| 类型 | 特征 | 推波助澜话术示例 |
|------|------|----------------|
| 经典情敌 | 明确抢男主 | "她不适合你，我才是最懂你的人" |
| 心机闺蜜 | 表面是女主朋友 | "我帮你看着他，你放心"（实际在勾引） |
| 幕后母亲 | 男主母亲操控全局 | "那个女人配不上我们家" |
| 伪弱者 | 装病/装可怜 | "我不想打扰你们，但我真的好害怕……" |
| 主动挑衅型 | 当面挑衅女主 | "他昨晚在我这里，你不知道吗？" |
| 致命犯罪型 | 策划谋杀/投毒/陷害 | 不靠话术，靠行动 |
| 寄生型养女 | 被收养后取代女主 | "爸爸说这些都是我的了" |
| 重生型情敌 | 带前世记忆的对手 | 精准避开上辈子的失败，重新布局 |
| 同父异母姐妹 | 私生女/同父异母 | "我才是爸爸真正的女儿" |

**对立方推波助澜通用话术**（在男主动摇时使用）：
- "让她冷静一下，追得太紧她更嚣张"
- "你越在乎她，她越拿捏你"
- "她不会真的走的，她离不开你"
- "她这是在逼你，你不能惯着"
- "你去找她就输了"

## 第二男主/新男主原型（五种）

**心理学底色**：安全型依恋（或"疗愈型回避"——自身有创伤但已整合）| 功能：为女主提供"安全基地"，不触发她的Fawn(讨好)模式，而是镜映她的真实自我

| 类型 | 特征 | 出场方式 |
|------|------|---------|
| 浪漫等待型 | 一直在等，女主走后才告白 | 在女主最低谷时出现 |
| 守护型青梅竹马 | 从小保护女主 | 女主回到家乡/老朋友圈 |
| 青梅竹马暗线型 | 童年有过交集，但双方不记得 | 通过信物/回忆碎片逐步揭开 |
| 暗黑浪漫型 | 危险/有权力/有创伤 | 女主被迫进入他的世界 |
| 命运安排型 | 偶然相遇，逐渐发展 | 职场/邻居/客户 |
