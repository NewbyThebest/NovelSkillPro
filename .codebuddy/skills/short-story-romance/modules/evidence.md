# 模块：物证链与情感道具

> 在第四步规划物证线、第五步创作时加载。

## 物证链（按冲击力递增）

| 层级 | 物证类型 | 示例 |
|------|---------|------|
| 习惯级 | 日常痕迹消失 | 牙刷没了/冰箱里没她做的菜/拖鞋不在门口 |
| 私人级 | 个人空间被清空 | 衣柜空了/化妆台只剩灰尘轮廓/书桌被搬走 |
| 信物级 | 象征物被遗弃 | 婚戒在桌角/项链留在首饰盒/情侣手链放在鞋柜上 |
| 文件级 | 法律/正式文件 | 签好的离婚协议/伴侣终止书/辞职信 |
| 终极级 | 不可逆铁证 | 胎儿死亡证明/她和别人的结婚证/知识产权归属确认书 |

**使用规则**：
- 在章纲中规划好每章出现/被发现的物证，确保全篇递增
- 每发现一个物证，角色反应必须是**身体动作**，不是"他感到震惊"
- 物证之间必须有间隔（不要一章内发现三个），给反应留空间

## 情感道具清单

| 类别 | 道具 | 功能 |
|------|------|------|
| 信物类 | 戒指/手链/项链/耳环/锡口哨 | 标记关系的存在或消亡 |
| 遗物类 | 骨灰盒/日记本/画册/素描本/缝纫机 | 连接已故角色 |
| 文件类 | 离婚协议/遗嘱/转让书/保险单/专利申请 | 推动法律/权力转移 |
| 食物类 | 栗子蛋糕/蓝莓蛋糕/妈妈的菜谱/凉掉的早餐 | 标记付出/被忽视 |
| 空间类 | 空衣柜/面朝下的相框/被换掉的照片/消失的牙刷 | 标记离开/被取代 |
| 声音类 | 录音/电话忙音/"您拨打的号码已停机" | 触发崩塌/揭露真相 |
| 童年类 | 时间胶囊/儿童画/贺卡/刻了名字的椅子 | 对比今昔 |

## 反转道具

女主给男主的"最后礼物"实际是终结关系的工具：

| 伪装 | 实际 |
|------|------|
| 周年礼物 | 离婚协议 |
| "你看看这个" | 遗嘱/知识产权声明 |
| 留在原位的信物 | 归还的终结标志（戒指在糖罐里/手链在鞋柜上） |
| "我帮你整理好了" | 她的东西全部搬走了 |
| 最后一顿饭 | 她做的最后一次他妈妈的菜谱 |
