# 模块：叙事模式与脱离方式

> 在第一步·核心设定脑暴时加载。用户可从中选择或组合。

## 叙事模式

| 模式 | 路线 | 适用题材 |
|------|------|---------|
| A 经典觉醒 | 隐忍→觉醒→离开→新生 | 都市情感/狼人短篇/医疗 |
| B 身份反转 | 低姿态→公开揭露→碾压 | 身份反转/狼人权力 |
| C 彻底消失 | 暗中准备→无声蒸发→远方重生 | 都市/狼人 |
| D 被动脱离 | 被外力切断→意外自由→拒绝回归 | 悬疑惊悚/科幻 |
| E 死后旁观/清算 | 已死→鬼魂/遗嘱→真相揭露→安息 | 鬼魂/家庭 |
| F 新王登基 | 旧关系崩→新伴侣→公开结合 | 狼人/都市爽文 |
| G1 重生复仇 | 前世记忆→惩罚仇人 | 重生/医疗 |
| G2 重生拯救 | 前世记忆→拯救对方→主动离开 | 重生/狼人 |
| H 代理复仇 | 替亲人复仇→完成使命 | 悬疑/双胞胎 |
| I 双线并行 | 两位女主同时觉醒 | 都市/闺蜜 |
| J 暗黑浪漫转型 | 离开旧爱→危险新男主→互相救赎 | 狼人长篇/暗黑浪漫 |

**组合示例**：A+F（觉醒后嫁新人）、B+F（身份反转+新王登基）、G1+H（重生+代理复仇）

## 脱离方式

| 方式 | 描述 | 适配模式 |
|------|------|---------|
| 安静计划型 | 暗中准备数周/数月，一切就绪后无声离开 | A/C |
| 身份抹除型 | 改名换姓，彻底消失，社交/法律身份全部清除 | C |
| 合同到期型 | 婚约/契约到期，以合法理由终止 | A/D |
| 被动触发型 | 因外力（失忆药/驱逐/死亡）被迫脱离 | D |
| 死亡型 | 女主已死/将死，通过遗嘱/鬼魂完成清算 | E |
| 新伴侣型 | 直接与新男主结合，用新关系替代旧关系 | F |
| 双线同步型 | 两位女主同时采取行动 | I |
| 身份替换型 | 双胞胎/姐妹冒充本人 | H |
| 自我牺牲型 | 为男主/他人牺牲后离开（捐器官/替罪/放弃权利） | G2 |
| 闪婚逃离型 | 冲动嫁给他人，以新婚姻完成断裂 | F |
| 权力夺取型 | 在公开场合夺取权力/地位，从被离开变成在上位 | B |
