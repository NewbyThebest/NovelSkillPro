# 角色状态追踪

## 使用规则

本文件是下一章直接读取的动态状态工作表，不是已登场角色名册或逐章历史。

只保留当前场景、下一章承接或当前阶段目标与约束仍需直接读取的角色。各字段覆盖更新，不按章节追加经过。

“最近更新依据”只记录最后一次实际改变角色状态的定稿章节。稳定事实写入角色设定，长期未决问题写入伏笔追踪，逐章过程写入定稿正文和十章章纲。

角色死亡、永久离场、完成场景作用，或其状态已不影响当前阶段创作时，删除对应条目；重新进入当前规划时再补回。
