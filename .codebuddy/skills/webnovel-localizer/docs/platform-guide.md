# 平台适配指南

> 确认目标平台后读取本文件，根据平台特性调整改写策略。

---

## Dreame

**背景**：字节跳动旗下Stary运营，以女性读者为主的社交阅读App。

**读者画像**：
- 18-35岁女性为主
- 偏好高情绪强度（emotional intensity）内容
- 通勤和睡前阅读为主，每次15-30分钟
- 主要来自美国、英国、菲律宾、印尼、尼日利亚

**热门品类**：Werewolf Romance #1，Mafia Romance #2，Billionaire Romance #3

**章节要求**：
- 每章 **800-1500词**（偏短）
- **前5章免费试读**，这5章决定读者是否付费 → 极其重要，必须在前5章内完成：
  - 建立核心冲突
  - 展示男女主的化学反应
  - 至少一个强烈的emotional moment
  - 每章结尾都有cliffhanger
- Cliffhanger在Dreame上比任何平台都重要——读者需要花coins解锁下一章

**写作调整**：
- 节奏要更快，铺垫更短
- 每章都是一个mini情感过山车
- 开头30词内必须有hook
- 避免纯世界观介绍章节——设定必须融入行动中

## GoodNovel

**背景**：掌阅海外版，读者性别分布较均衡。

**读者画像**：
- 18-40岁，男女比例约4:6
- 接受稍深度的内容
- 愿意在情节铺垫上多投入一点耐心

**热门品类**：Mafia Romance、Billionaire Romance、Werewolf都有市场。CEO/Billionaire类在GoodNovel上特别强。

**章节要求**：
- 每章 **1500-2000词**（中等）
- 试读章节同样重要但容忍度略高
- 可以有稍多的世界观介绍，但仍需融入情节

**写作调整**：
- 可以比Dreame稍慢一点展开
- 角色心理描写可以更细腻
- 支线情节的空间更大
- 仍需保持每章有hook和cliffhanger

**签约特点**：
- 签约作者待遇相对优厚
- 编辑反馈机制较好

---

## Webnovel

**背景**：阅文集团海外版，品类最广，同时有男频和女频。

**读者画像**：
- 品类最多样化的读者群
- 包含大量从中文网文翻译作品培养起来的读者
- 对世界观构建的容忍度更高
- 部分读者习惯了中式网文风格

**热门品类**：覆盖面最广——Werewolf、Mafia、Fantasy、System/Reincarnation都有

**章节要求**：
- 每章 **1500-2500词**（较长）
- 可以有更长的世界观铺垫
- 仍需注意章节结构

**写作调整**：
- 世界观构建可以更详细
- 可以接受稍复杂的设定体系
- 但language style仍需地道——不要因为Webnovel对设定容忍度高就放松语言要求
- 对话仍需短促有力

**签约特点**：
- 采用「邀请制」签约，编辑主动筛选
- 反馈机制较慢，作者可能长期等待
- 全勤奖和日更奖励

## 平台选择速查

| 维度              | Dreame         | GoodNovel         | Webnovel   |
| ----------------- | -------------- | ----------------- | ---------- |
| 章节字数          | 800-1500       | 1500-2000         | 1500-2500  |
| 节奏要求          | 最快           | 中等              | 稍慢可接受 |
| 世界观深度        | 最浅，融入行动 | 中等              | 可更深     |
| Cliffhanger重要性 | 极高           | 高                | 高         |
| 前5章要求         | 极严格         | 严格              | 较严格     |
| 最热品类          | Werewolf       | Mafia/Billionaire | 最广泛     |
| 读者耐心          | 最低           | 中等              | 较高       |

---

## 跨平台通用原则

无论哪个平台，以下原则不变：
1. 第一句话就要有力量
2. 每章必须有cliffhanger
3. 语言必须地道（参照 style-guide.md）
4. 品类设定必须到位（参照对应品类文件）
5. 女主不能纯粹被动
6. 必须有HEA/HFN
