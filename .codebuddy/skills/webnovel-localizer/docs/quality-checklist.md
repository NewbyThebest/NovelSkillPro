# 质量检查清单

> 每次改写完成后，逐项对照本清单检查。全部通过后才能输出给操作者。

---

## 语言风格（必检）

- [ ] **翻译腔清零**：是否消除了所有「翻译腔」表达？对照 style-guide.md 替换表逐一排查。
- [ ] **Deep POV统一**：是否全篇保持统一的叙事视角？一个场景是否只有一个POV角色？是否存在POV角色不可能知道的信息？
- [ ] **Show不Tell**：是否用身体反应/动作/内心独白展示情感，而非直接陈述？搜索「felt」「was very」「seemed」等tell标志词。
- [ ] **对话短促**：对话是否短促有力？是否使用了缩写（don't, can't, won't）？是否有角色在「演讲」？
- [ ] **五感描写**：每个场景是否至少涵盖3种感官？是否有触觉和嗅觉描写（不仅仅是视觉）？
- [ ] **斜体内心独白**：是否使用了斜体碎片化的内心独白？是否自然融入叙事？

---

## 章节结构（必检）

- [ ] **Opening Hook**：章节开头50词内是否有强力hook？第一句话是否有冲击力？是否避免了「天气描写」「起床」等弱开头？
- [ ] **Cliffhanger Ending**：章节是否在最紧张的瞬间断章？读者是否有强烈动机点开下一章？
- [ ] **章节词数**：是否符合目标平台要求？（Dreame: 800-1500 / GoodNovel: 1500-2000 / Webnovel: 1500-2500）
- [ ] **节奏流畅**：是否每个段落都有forward momentum？是否有拖沓的纯描写段落？

---

## 品类设定（按品类检查）

### Mafia Romance
- [ ] **家族体系到位**：是否有清晰的家族权力结构？职位称谓是否准确？
- [ ] **Trope明确**：核心Trope是否清晰可辨？（Arranged Marriage / Enemies to Lovers / 等）
- [ ] **男主原型**：是否符合Dark Possessive Hero特质？是否morally gray但对女主有soft spot？
- [ ] **女主不被动**：女主是否有feisty/不屈服的特质？是否敢于挑战男主？
- [ ] **文化细节**：是否有Omertà/loyalty/family等文化概念的自然融入？

### Werewolf Romance
- [ ] **Pack体系到位**：Alpha/Luna/Beta/Omega等级是否清晰？是否避免了「宗门化」？
- [ ] **Mate Bond呈现**：命定伴侣纽带是否有足够的情感冲击力？是否通过气味/触碰/等具体方式呈现？
- [ ] **Inner Wolf声音**：是否有斜体的狼人格对话？（*Mine.* / *Mate.* 等）
- [ ] **设定简洁**：等级体系是否简洁（不超过5-6个等级）？世界观是否服务于romance而非反客为主？
- [ ] **Rejection句式**（如适用）：是否使用了标准句式 "I, [name], reject you as my mate"？

---

## 输出格式（必检）

- [ ] **词数标注**：是否标注了每章/段的词数？
- [ ] **POV标注**：是否标注了POV角色？
- [ ] **改写说明**：关键改写处是否有 `【改写说明】` 注释？是否解释了为什么这样改？
- [ ] **学习要点**：结尾是否附了「本次改写学习要点」3-5条？

---

## 最终确认

- [ ] **HEA/HFN走向**：整体情感走向是否指向HEA或HFN？
- [ ] **无红线违规**：男主是否没有对女主施加身体暴力？是否没有美化真实犯罪？
- [ ] **平台适配**：是否针对目标平台做了相应调整？（参照 platform-guide.md）
