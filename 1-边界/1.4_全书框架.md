# 1.4 全书框架与节奏拆解

## 2. 核心节奏公式

### 标准单元循环

本书最适合使用的核心循环是：

> 被定价为废物 -> 被推入规则局 -> 观察漏洞 -> 承残技 / 收样本 -> 融合求生 -> 外界误判 -> 身份小幅跃迁 -> 更高层势力重新定价

这个循环可以反复使用，但每轮必须改变三件事：

- **局的类型要升级**：从热场求生，到低级拳赛，到买手验证，到交易转运，到城市暗线追杀。
- **能力材料要升级**：从弱者残技，到废弃拳手样本，到改造失败品，到门派武技碎片，到黑市研究数据。
- **身份误判要升级**：从“运气好活下来”，到“隐藏天赋苗子”，到“可交易样本”，再到“无法被现有评级解释的危险变量”。

### 开篇黄金三章公式

> 现实诱饵 -> 骗局收网 -> 兄弟分岔 -> 死局上台 -> 金手指异动 -> 残技承载 -> 临时融合 -> 活命但不暴露

前三章的重点不是打赢强敌，而是让读者相信：

- 祁临真的处在会死的局里。
- `天衍武藏` 有明确限制，不是看一眼学会。
- 弱者残技也能成为爽点材料。
- 园区没有看穿真相，后续藏拙空间成立。

### 中期多线公式

> 明线拳赛保命 + 暗线样本收集 + 兄弟线互救 + 园区线试探 + 买手线定价

中期不能只连续打拳。每个拳赛单元都要同时推进至少两条线：

- 打赢或活过这一场，解决即时危机。
- 获得一个能力样本、规则情报、身份权限或江照野线索。
- 让某个势力对祁临产生新的误判或兴趣。

### 爽点落袋公式

> 压迫够重 -> 限制说清 -> 弱材料看似无用 -> 祁临组合出活法 -> 对手和观众误读 -> 读者知道真实收益

本书的爽点不宜写成单纯碾压。更适合写“读者看懂祁临在规则缝隙里做了什么”，外界却看错、看轻或看不懂。

## 3. 换地图逻辑分析

### 什么时候换地图

本书的换地图不应按战力等级机械推进，而应由“资源枯竭 + 身份暴露风险 + 利益链上移”共同驱动。

| 换地图节点 | 换前问题 | 换后收益 | 注意事项 |
|:---|:---|:---|:---|
| 热场 -> 低级拳池 | 热场是一次性死局，不能长期重复 | 主角获得最低级身份和行动半径 | 不要让身份跃迁太轻松，必须留下园区怀疑。 |
| 低级拳池 -> 园区核心区 | 低级对手和残技样本开始重复 | 接触苗子营、医疗区、盘口后台 | 新区域必须继续服务求生和藏拙，不要变成设定观光。 |
| 园区 -> 地下武馆 / 买手网络 | 园区内部压迫接近天花板 | 展开门派、财团、黑市能力交易 | 换地图前要先让园区背后势力露头，不能突兀离开。 |
| 海外黑市 -> 国内都市高武 | 海外线容易越写越黑、越写越脱离都市感 | 拉回武道公司、武馆、城市秩序、熟人社会 | 国内线要保留危机感，不能变成普通校园或武考路线。 |
| 国内表层 -> 新江湖深层 | 单个公司或武馆已不足以承载终局 | 进入门派财团核心和能力交易源头 | 后期核心不是堆组织名，而是回答“谁有资格定义人的价值”。 |

### 换地图前兆

- 当前地图的拳赛、敌人、能力样本出现重复感。
- 祁临在当前身份下已经拿到主要权限，继续留下只会拖节奏。
- 园区或势力对祁临的误判升级，旧伪装快撑不住。
- 江照野线索指向更高层控制者。
- 买手、门派外门、财团武装或黑市研究所主动伸手。

### 换地图禁忌

- 不为扩大世界观而换地图。
- 不在开篇三章塞门派、财团、黑市研究所全貌。
- 不让主角刚活过第一场就立刻逃出园区，否则园区求生线的爽点会被浪费。
- 不把国内阶段写成传统武考打脸，国内地图必须承接境外黑产回流和都市新江湖暗线。

## 4. 对新书的启示

### 开局节奏

- 第一章从骗局收网后切入：车门上锁、护照被收、合同作废、园区显形。
- 江照野和祁临必须在第一章完成命运分岔：一个被当资产，一个被当耗材。
- `天衍武藏` 第一章只给异动，第二章承技，第三章融合兑现。
- 第一场不能赢得漂亮，要赢得危险：祁临活下来，但外界只看到异常，不知道本质。

### 中期布局

- 园区阶段至少要写出三个层级：低级拳池、苗子营、园区后台 / 医疗改造区。
- 江照野不是背景板，他的苗子营线要定期回扣主线，成为祁临冒险的情感理由和利益筹码。
- 能力收集必须有代价：承载冲突、身体负担、样本污染、外界怀疑，都可以作为中期压力。
- 反派和管事的行动必须有商业逻辑：验证价值、控制风险、提高盘口、出售样本，而不是单纯找主角麻烦。

### 后期格局

- 后期不是“复制万物”，而是“万技归藏”：祁临把残技样本融合成自己的武道。
- 园区只是第一层恶，真正的敌人是把天赋、身体、拳手和能力样本明码标价的新江湖秩序。
- 主角身份应从耗材、拳手、黑马、交易品、危险变量，最终变成自开道路者。
- 终局爽点应落在“他不再被任何检测仪、评级表、盘口或买手定义”。

### 对后续阶段的约束

- `2-设定` 阶段需要细化 `天衍武藏` 的承载上限、融合代价、样本污染与外界误判机制。
- `3-大纲` 阶段需要把第一卷重点放在园区身份跃迁，不要过早逃离园区。
- `4-正文` 阶段前三章必须保证强危机、兄弟分岔、金手指首爆与藏拙误判全部落地。
- 任意阶段如果出现传统修仙、武考打脸、普通电诈纪实、无脑反派或主角首战碾压，都应回退修正。

## 本阶段结论

当前全书框架的核心不是“主角一路复制能力变强”，而是：

> 祁临被高武黑色产业链判为耗材，却从耗材堆的残技里拼出活路；他每一次身份跃迁，都是对园区定价体系的一次反证。

下一步 `1.5_微观节奏拆解.md` 应重点拆两类样本：

- 开篇前三章的强危机、分岔、承技、融合、误判。
- 一个完整的小高潮单元，例如“第二场验证局”或“伤员区样本争夺”，拆清起承转合、情绪积蓄和爽点落袋。
